#산업_신기술 #인공지능

## 정의
신경망분석 (Artificial Neural Network)
인간의 뇌 구조인 뉴런 간의 연결망을 모방하여 데이터를 학습하고 패턴을 인식하는 분석 기법, 신경망분석
- 입력층, 은닉층, 출력층의 다층 구조를 통해 복잡한 비선형 문제를 해결하는 기법
- 학습 과정을 통해 각 연결의 가중치(Weight)를 조정하며 예측 오차를 최소화하는 분석 활동
특징) 자기 학습 능력, 비선형 모델링, 유연한 구조, 블랙박스 특성

## 목적
- 이미지 인식, 음성 처리, 자연어 이해 등 전통적인 방법으로 해결하기 어려운 비정형 데이터 분석
- 시계열 예측, 금융 시장 분석, 질병 진단 등 정밀한 예측 모델 구축
- 대규모 데이터로부터 특징(Feature)을 자동 추출하여 분석의 효율성 및 정확도 제고
- 강화학습 등과 결합하여 복잡한 환경에서의 최적의 의사결정 모델 구현

## 구성요소
- 계층 구조: 입력층(Input Layer), 은닉층(Hidden Layer), 출력층(Output Layer)
- 구성 요소: 가중치(Weight), 편향(Bias), 넷 입력 함수(가중합 계산)
- 활성화 함수(Activation Function): 단조 증가 함수로서 비선형성 부여 (Sigmoid, ReLU, Step, Tanh)
- 학습 알고리즘: 오차 역전파(Backpropagation), 경사 하강법(Gradient Descent)
- 유형: CNN(이미지), RNN(순차 데이터), DNN(심층 신경망), GAN(생성형)

## 활성화 함수 특징
- Sigmoid: 0~1 사이 출력, 기울기 소실(Vanishing Gradient) 문제 존재
- ReLU: 0보다 작으면 0, 크면 x 출력, 연산 속도가 빠르고 기울기 소실 문제 완화
- Softmax: 다중 분류 시 출력층에서 확률값으로 변환하여 사용

## 기대효과
- 고차원의 복잡한 데이터 분석 정확도 비약적 향상
- [[Datamining]]의 핵심 알고리즘으로 활용되어 인공지능 성능 고도화

## 키워드
- 뉴런, 가중치, 오차 역전파, 활성화 함수, 딥러닝

## 암기법
- 입은활출: 입력층, 은닉층, 활성화함수, 출력층
- 지비강: 지도학습, 비지도학습, 강화학습
