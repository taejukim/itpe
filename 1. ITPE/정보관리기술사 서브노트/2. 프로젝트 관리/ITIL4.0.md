## 정의
ITIL4.0 (IT Infrastructure Library)
[[ITSM]] 구축을 위한 Best Practice 모음집, ITIL의 정의
- IT 서비스 관리의 사실상의 산업표준
- 비즈니즈 연속성을 위해 안정적인 서비스가 가능하도록 설계된 관리 기법 BP 모음

## 구성도
- 서비스 관리의 4차원 모델
- 조직과 사람들 : 이해 관계자 중요성
- 정보 및 기술 : 정보의 중요성과 가용성, 클라우드 등 기술의 중요성
- 파트너 및 공급업체 : 공급업체, SIAM(Service Integration and Management) 프레임워크 사용
- 가치 흐름과 프로세스 : 가치 창출을 위한 프로세스

## 구성요소
* SVS(Service Value System) 
	* Guiding Principle : 지도 원칙
	* Governance : 거버넌스(EDM 모델, Direct(지시) > Monitoring(모니터링) > Evaluate(평가))
	* SVC(Service Value Chain) : 서비스 가치 사슬
	* Practice : 실행
	* Continual Improvement : 지속적인 개선
* SVC(Service Value Chain)
	* Plan : 계획
	* Engage : 참여
	* Design/Transition : 설계/이행
	* Obtain/Build : 획득/구축
	* Delivery/Support : 인도/지원
	* Product/Service : 제품과 서비스
	* Improvement : 개선
## 관리 수행
- 일반 관리 : 전략, 아키텍쳐, 위헙관리
- 서비스 관리 : SLM, 가용성, 서비스 데스트
- 기술 관리 : 배포, 인프라 플랫폼, SW개발 관리

## 키워드
* SVS, SVC, 4차원 모델
