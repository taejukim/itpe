#정보관리기술사 #디지털서비스 #앰비언트컴퓨팅 #IoT #AI #정리

## 정의
일상에 자연스럽게 스며든 지능형 컴퓨팅, 앰비언트 컴퓨팅(Ambient Computing)
- IT 기기가 사용자의 일상에 자연스럽게 자리잡아, 인지되지 못한 상태로 활용되는 환경 지능형 컴퓨팅 개념
- IoT 디바이스를 통해 대상의 상태를 추적·감지하여 상황을 이해하고, 필요한 정보 및 요구를 예측·지원하여 적시적으로 제공하는 기술
## 키워드
* UI/UX, AI, 5G, VR/AR, IoT, 센서, 아마존 고, 앰비언트 인비저블 인텔리전스, 121회 기출
## 암기법
* UI AI 5V (UI/UX, AI, 5G, VR/AR) - 앰비언트 컴퓨팅 4대 핵심 기술
## 특징
- 비가시성(Invisible): 사용자가 기기의 존재를 인지하지 못한 상태에서 자연스럽게 동작
- 상황 인식(Context-Aware): 센서·AI 기반으로 사용자 환경과 상태를 실시간 추적·감지
- 선제적 서비스: 사용자의 요구를 예측하여 적시적으로 정보 및 서비스 제공
- 멀티모달 인터페이스: 음성, 제스처, 시선 등 다양한 인터랙션 방식 지원
## 목적
- 기술과 일상의 자연스러운 융합으로 사용자 경험 극대화
- 사용자의 명시적 조작 없이 상황에 맞는 지능형 서비스 제공
## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 인지 | IoT 센서/디바이스 | 환경 상태 추적·감지 |
| 연결 | 5G/Wi-Fi 통신 | 초저지연 실시간 데이터 전송 |
| 판단 | AI/ML 엔진 | 상황 이해 및 요구 예측 |
| 인터페이스 | UI/UX, VR/AR | 자연스러운 사용자 인터랙션 |
| 서비스 | 클라우드 플랫폼 | 데이터 처리 및 서비스 운영 |
