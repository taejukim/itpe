#정보관리기술사 #디지털서비스 #자율주행 #정리

## 정의
**인지·판단·제어 기반 자율 운행 차량, Smart Car(자율주행)**
- 센서, 카메라, 레이더, 초음파 등으로 주행 환경을 인지하고, AI 기반으로 판단·제어하여 운전자 개입 없이 자율 운행하는 차량
- NHTSA(0~4단계), SAE(0~5단계) 기준으로 자율주행 레벨을 구분하여 단계적 발전
## 키워드
* SAE 레벨, NHTSA, ADAS, LiDAR, V2X, CAN, LIN, ADR, 인지-판단-제어
## 암기법
* 인판제 (인지, 판단, 제어) - 자율주행 3대 핵심 기술
## 특징
- 자율주행 레벨 체계: SAE 0~5단계로 자율주행 수준 정의
- 다중 센서 융합: 카메라, 레이더, LiDAR, 초음파 센서 통합 활용
- V2X 통신: 차량-차량(V2V), 차량-인프라(V2I), 차량-보행자(V2P) 통신
- ADR 기록: Autonomous Data Recorder를 통한 자율주행 데이터 기록·관리
## 목적
- 교통사고 감소 및 도로 안전성 향상
- 이동의 편의성 증대 및 교통 효율 최적화
## 구성요소
- 인지: 센서(LiDAR, Radar), 카메라, 초음파, GPS/GNSS
- 판단: AI/ML 알고리즘, HD Map, 경로 계획, 상황 인식
- 제어: 조향, 가감속, 제동, ECU
- 통신 I/F: LIN(저속), CAN(ISO 11898, 고속), V2X
- 데이터: ADR(Autonomous Data Recorder)
## 자율주행 레벨
- Level 0: 비자동화 (운전자가 모든 조작 수행)
- Level 1: 운전자 보조 (가감속 또는 조향 중 하나)
- Level 2: 부분 자동화 (가감속 + 조향 동시 지원)
- Level 3: 조건부 자동화 (시스템 운전 + 운전자 적절 개입)
- Level 4: 고도 자동화 (특정 주행모드에서 모든 운전 조작 수행)
- Level 5: 완전 자동화 (모든 조건에서 항상 자율주행)
## 구성도
```
[자율주행 아키텍처]
  센서/카메라  →  인지(Perception)  →  판단(Decision)  →  제어(Control)
  LiDAR,Radar      환경인식              경로계획            조향/가감속
  초음파,GPS       객체감지              상황판단            제동/ECU

[자율주행 레벨] (SAE 기준)
  L0(비자동) → L1(보조) → L2(부분) → L3(조건부) → L4(고도) → L5(완전)
  NHTSA: 0~4단계 / SAE: 0~5단계

[통신 I/F]
  LIN(저속) ─── CAN(ISO 11898, 고속) ─── V2X(차량-외부)
```
