#정보관리기술사 #디지털서비스 #클라우드보안 #SECaaS #정리

## 정의
클라우드 기반 보안 서비스 모델, SecaaS
- 보안 기능을 클라우드 서비스 형태로 제공하는 클라우드 보안 서비스
- CSP 및 전문 보안서비스 제공자(SSP)가 제공하며, CASB를 통한 독립적 보안 브로커링 가능
## 키워드
* CSP SECaaS, SSP SECaaS, CASB, IAM, DLP, BCDR, CSA 12영역
## 암기법
* 식데침암네업: (CSP SECaaS) 식별/접근관리, 데이터손실방지, 침입관리, 암호화, 네트워크보안, 업무연속성/재난복구
- 취웹이지 이보: (SSP SECaaS) 취약성탐지, 웹보안, 이메일보안, 지속적감시, 이벤트관리, 보안접근
- 접데로이 위암D: (CASB 기능) 접근통제, 데이터손실방지, 로깅/감사, 이상탐지, 위험평가, 암호화, DRM
- 네웹신지침S 취암이D업보: (CSA 12영역)
## 특징
- 서비스형 보안: 별도 보안 장비 도입 없이 구독형 보안 서비스 이용
- 이중 제공 구조: CSP(서비스 제공자)와 SSP(전문 보안업체) 이원화
- CASB 브로커링: 독립적 보안 중개를 통한 가시성/통제력 확보
- CSA 표준: CSA 12가지 보안 영역 기반 체계적 보안 서비스 분류
## 목적
- 클라우드 환경의 보안 위협에 대한 전문적 대응 체계 확보
- 보안 인프라 투자 비용 절감 및 최신 보안 기술 즉시 적용
## 구성요소
| 구분 | 유형 | 서비스 항목 |
|------|------|-------------|
| CSP SECaaS | 서비스 제공자 보안 | 식별/접근관리(IAM), 데이터손실방지(DLP), 침입관리(IM), 암호화, 네트워크보안, 업무연속성/재난복구(BCDR) |
| SSP SECaaS | 전문 보안업체 보안 | 취약성탐지, 웹보안, 이메일보안, 지속적감시, 이벤트관리, 보안접근 |
| CASB | 보안 브로커 | 접근통제, 데이터손실방지, 로깅/감사, 이상탐지, 위험평가, 암호화, DRM |
## 구성도
```
[클라우드 사용자]
       │
    [CASB] ─── 접근통제/이상탐지/DLP/암호화
       │
  ┌────┴────┐
  ▼         ▼
[CSP]     [SSP]
 │         │
 │  식데침  │  취웹이
 │  암네업  │  지이보
 ▼         ▼
[클라우드 인프라/서비스]
```
