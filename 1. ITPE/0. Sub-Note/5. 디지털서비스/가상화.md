## 정의
물리적 자원의 논리적 통합/분할 기술, 가상화
- 물리적 하드웨어 자원을 논리적으로 추상화하여 하나의 자원을 여러 개처럼, 또는 여러 자원을 하나처럼 사용할 수 있게 하는 기술
- 서버, 스토리지, 네트워크, 데스크톱 등 IT 인프라 전반에 적용
## 키워드
* 서버 가상화, OS 가상화, 스토리지 가상화, 네트워크 가상화, VDI, 하이퍼바이저, 전가상화, 반가상화
## 암기법
* 앱서OS스데네 (Application, 서버, OS, 스토리지, Desktop, 네트워크)
## 특징
- 자원 효율성: 물리적 자원의 활용률을 극대화하여 비용 절감
- 격리성(Isolation): 가상 환경 간 독립적 운영으로 보안성/안정성 확보
- 유연한 확장: 논리적 자원 할당으로 신속한 프로비저닝 가능
- 이식성: 하드웨어 독립적으로 가상 머신 이동/복제 가능
## 구성요소
| 유형 | 대표 기술 | 설명 |
|------|-----------|------|
| Application 가상화 | SaaS | 응용 프로그램 단위 가상화 제공 |
| 서버 가상화 | OS + HW | 하나의 물리 서버에서 다수 가상 서버 운영 |
| OS 가상화 | VMM, 하이퍼바이저, Docker, LXC, LXD | 전가상화, 반가상화, OS레벨 가상화 |
| 스토리지 가상화 | EMC, 베리타스 | 이기종 스토리지를 Pool로 LUN 관리 |
| Desktop 가상화 | VDI, SBC, Zero PC | PC 가상화를 통한 원격 데스크톱 제공 |
| 네트워크 가상화 | VLan, SDN | 물리 네트워크의 논리적 분리/통합 |
## 구성도
```
┌─────────────────────────────────────────────────┐
│              Application 가상화 (SaaS)           │
├─────────────────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────────────────┐ │
│  │ Desktop 가상화│  │     서버 가상화           │ │
│  │ VDI, SBC     │  │  VM  │  VM  │  VM        │ │
│  └──────────────┘  ├──────────────────────────┤ │
│                    │  OS 가상화 (Hypervisor)   │ │
│                    │  전가상화 / 반가상화       │ │
│                    ├──────────────────────────┤ │
│                    │     Hardware (CPU/Mem)    │ │
│                    └──────────────────────────┘ │
├─────────────────────────────────────────────────┤
│  스토리지 가상화        │  네트워크 가상화        │
│  (Pool → LUN 관리)     │  (VLan, SDN)          │
└─────────────────────────────────────────────────┘
```
