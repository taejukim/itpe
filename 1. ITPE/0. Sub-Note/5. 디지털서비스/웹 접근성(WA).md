#정보관리기술사 #디지털서비스 #웹접근성 #WCAG #전자정부

## 정의
장애인·고령자 포함 모든 사용자의 웹 이용 보장, 웹 접근성(WA)
- 장애인, 고령자 등을 포함한 모든 사용자가 웹 콘텐츠에 동등하게 접근하고 이용할 수 있도록 보장하는 웹 품질 기준
- 인식의 용이성, 운용의 용이성, 이해의 용이성, 견고성 4대 원칙 기반
## 키워드
* 인운이견, WCAG, 인식의 용이성, 운용의 용이성, 이해의 용이성, 견고성, 전자정부 품질관리
## 암기법
* 인운이견 (인식의 용이성, 운용의 용이성, 이해의 용이성, 견고성)
- 대멀명 (대체 텍스트, 멀티미디어 대체, 명료성) → 인식
- 키충광쉬 (키보드 접근성, 충분한 시간, 광과민성, 쉬운 네비게이션) → 운용
- 카예컨입 (가독성, 예측 가능성, 콘텐츠 논리성, 입력 도움) → 이해
- 문어 (문법 준수, 웹 애플리케이션 접근성) → 견고성
- 호접개접편효신 (전자정부 품질관리 7원칙)
## 특징
- 인식의 용이성(Perceivable): 대체 텍스트, 멀티미디어 대체 수단, 명료성 제공
- 운용의 용이성(Operable): 키보드 접근성, 충분한 시간, 광과민성 예방, 쉬운 네비게이션
- 이해의 용이성(Understandable): 가독성, 예측 가능성, 콘텐츠 논리성, 입력 도움
- 견고성(Robust): 문법 준수, 웹 애플리케이션 접근성 보장
## 목적
- 정보 접근의 평등권 보장 및 디지털 격차 해소
- 전자정부 웹사이트 품질 확보 및 법적 의무 준수
## 구성요소
| 원칙 | 세부 지침 | 암기 | 설명 |
|------|-----------|------|------|
| 인식의 용이성 | 대체 텍스트, 멀티미디어 대체, 명료성 | 대멀명 | 비텍스트 콘텐츠에 대한 대안 제공 |
| 운용의 용이성 | 키보드 접근성, 충분한 시간, 광과민성 예방, 쉬운 네비게이션 | 키충광쉬 | 다양한 입력장치 및 시간 보장 |
| 이해의 용이성 | 가독성, 예측 가능성, 콘텐츠 논리성, 입력 도움 | 카예컨입 | 콘텐츠의 이해 및 오류 방지 |
| 견고성 | 문법 준수, 웹 앱 접근성 | 문어 | 웹 표준 준수 및 보조기기 호환 |
## 전자정부 웹사이트 품질관리 원칙
| 원칙 | 설명 |
|------|------|
| 호환성 | 웹 표준 준수 |
| 접근성 | 인운이견 4대 원칙 준수 |
| 개방성 | 검색엔진, 검색로봇 접근 보장 |
| 접속성 | 응답속도, 용량 최적화 |
| 편의성 | 안내, 매뉴얼 제공 |
| 효율성 | 페이지수, 방문자수 관리 |
| 신뢰성 | 최신화, 지식재산권, 개인정보 보호 |
