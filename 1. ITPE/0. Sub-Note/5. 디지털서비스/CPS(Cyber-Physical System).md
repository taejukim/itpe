#정보관리기술사 #디지털서비스 #CPS #스마트팩토리 #IoT

## 정의
물리-사이버 융합 시스템, CPS
- 물리적인 실제 시스템과 IoT 플랫폼을 연결하여 실시간으로 제어, 적용, 예측하는 시스템
- 센서·액추에이터로 물리 세계를 감지하고, 네트워크·컴퓨팅으로 분석·제어하는 Industry 4.0의 핵심 기술
## 키워드
* Cyber-Physical System, 센서, 액추에이터, 네트워크, 컴퓨팅
* 디지털트윈, IoT, 실시간 제어, 피드백 루프, 임베디드 시스템
## 암기법
* CPS 구성: 센네컴액 (센서-네트워크-컴퓨팅-액추에이터)
* CPS = 물리(Physical) + 사이버(Cyber) + 시스템(System)
## 특징
- ==실시간 연동==: 물리 세계와 사이버 세계의 실시간 양방향 통신
- ==자율 제어==: 데이터 분석 기반 자동 피드백·제어
- ==예측 기능==: 시뮬레이션·AI 기반 미래 상태 예측
- ==확장성==: 다양한 물리 시스템과 연동 가능
## 목적
- 물리 시스템의 실시간 모니터링·최적화·자율 제어 실현
- 제조·에너지·교통 등 산업 전반의 지능화
## 구성요소
- ==센서(Sensor)==: 물리 환경 데이터 수집 (온도, 압력, 진동 등)
- ==네트워크(Network)==: 데이터 전송 통신 인프라 (유·무선, IoT 프로토콜)
- ==컴퓨팅(Computing)==: 데이터 분석·처리·의사결정 (엣지·클라우드)
- ==액추에이터(Actuator)==: 물리 시스템 제어·조작 (모터, 밸브 등)
- ==디지털 트윈(Digital Twin)==: 물리 객체의 가상 복제본
- ==미들웨어==: 이기종 시스템 간 데이터 연동 플랫폼
## 구성도
```
[CPS 동작 구조]

 ┌────────── Cyber World ──────────┐
 │  ┌──────────┐  ┌──────────────┐ │
 │  │ 데이터    │  │  AI/분석     │ │
 │  │ 수집·저장 │→ │  의사결정    │ │
 │  └──────────┘  └──────┬───────┘ │
 └───────────────────────┼─────────┘
          ↑ 센서          │ 제어
          │ 데이터         ▼ 명령
 ┌────────┴────── Physical World ──┐
 │  ┌──────────┐  ┌──────────────┐ │
 │  │  센서     │  │ 액추에이터   │ │
 │  │ (감지)    │  │  (제어)      │ │
 │  └──────────┘  └──────────────┘ │
 │        물리적 환경 (공장, 설비)    │
 └─────────────────────────────────┘

         ↕ 실시간 피드백 루프
```
