#정보관리기술사 #디지털서비스 #NFT #블록체인 #토큰 #정리

## 정의
복제 불가 블록체인 기반 암호 토큰, NFT
- 하나의 토큰을 다른 토큰으로 대체하는 것이 불가능하며 특정 디지털 자산을 나타내는 복제가 불가능한 블록체인 기반의 암호 토큰
- Non-Fungible Token, 고유한 식별값을 가지며 디지털 자산의 소유권과 진위를 증명하는 대체불가 토큰
## 키워드
* ERC-20, ERC-721, ERC-1155, NFT거래소, 스마트계약, 토큰이코노미, 메타버스, 예술품
## 암기법
* ERC 구분: ERC-20(대체가능, FT), ERC-721(대체불가, NFT 단일), ERC-1155(대체가능+불가 혼합)
## 특징
- 대체불가성: 각 토큰이 고유한 식별값을 가져 1:1 대체 불가(Non-Fungible)
- 소유권 증명: 블록체인에 소유권 이력이 투명하게 기록·추적
- 상호운용성: 다양한 플랫폼과 마켓플레이스 간 거래 가능
- 프로그래밍 가능: 스마트 계약 기반 로열티 자동 지급 등 설정
## 목적
- 디지털 자산의 원본 진위 증명 및 소유권 보호
- 창작자 경제(Creator Economy) 활성화 및 디지털 자산 거래 생태계 구축
## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 토큰 표준 | ERC-20 | Fungible Token, 대체가능 토큰 표준 |
| 토큰 표준 | ERC-721 | Non-Fungible Token, 대체불가 토큰 표준 |
| 토큰 표준 | ERC-1155 | Multi Token, FT+NFT 혼합 표준 |
| 기반기술 | 스마트 계약 | 거래 조건 자동 실행 계약 |
| 기반기술 | 이더리움 블록체인 | ERC 표준 토큰 발행 플랫폼 |
| 생태계 | NFT 거래소 | OpenSea 등 마켓플레이스 |
| 활용 | 메타버스 | 가상 세계 디지털 자산 |
| 활용 | 디지털 예술품 | 디지털 아트 원본 증명 |
## 구성도
```
[창작자] ──발행──▶ [스마트 계약(ERC-721/1155)]

                           │
                 ┌─────────┴─────────┐
                 ▼                   ▼
          [NFT 거래소]         [메타버스 플랫폼]

                 │                   │
                 └─────────┬─────────┘
                           ▼
                    [구매자/소유자]
                    (소유권 블록체인 기록)

[ERC 표준] Ethereum Request for Comments
 ERC-20(FT) ── ERC-721(NFT) ── ERC-1155(Multi)
```
