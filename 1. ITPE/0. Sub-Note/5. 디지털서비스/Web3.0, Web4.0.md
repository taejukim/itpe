#정보관리기술사 #디지털서비스 #Web3 #Web4 #시맨틱웹 #정리

## 정의
**개인 맞춤형 지능 서비스 플랫폼, Web 3.0/4.0**
- Web 3.0: 다수의 컴퓨터를 Seamless 네트워킹으로 연결하여 인간 중심의 정보 서비스를 제공하는 개인 맞춤형 서비스 플랫폼
- Web 4.0: 인간과 기계가 공생하는(Symbiotic) 웹으로, 모바일 웹 기반 All Device에서 가상/현실 세계를 연결하는 차세대 웹
## 키워드
* Semantic Web, 지능형 Agent, 상황인식 컴퓨팅, 의미기반 검색엔진, Big Data, 클라우드, Symbiotic Web
## 암기법
* 개소융 (개인화, 소통, 융합) - Web 3.0 핵심 가치
## 특징
- Semantic Web: 기계가 이해할 수 있는 의미 기반 웹 데이터 구조
- 지능형 Agent: 사용자 맥락을 이해하고 능동적으로 서비스를 제공하는 소프트웨어 에이전트
- 상황인식 컴퓨팅: 사용자의 상황·맥락을 인식하여 적합한 서비스 제공
- 의미기반 검색엔진: 키워드가 아닌 의미와 맥락을 이해하는 검색 기술
## 목적
- 인간 중심의 개인 맞춤형 지능 정보 서비스 제공
- 가상과 현실의 융합을 통한 새로운 사용자 경험 창출
## Web 진화 단계
- Web 1.0: 정보 제공 중심, 일방향, 읽기 전용 웹
- Web 2.0: 참여·공유·개방, 양방향, 플랫폼 웹
- Web 3.0: 개인화·소통·융합, Semantic Web, 지능형 서비스 플랫폼
- Web 4.0: 인간-기계 공생(Symbiotic), 모바일 웹, All Device, 가상/현실 연결
## 구성요소
- Semantic Web: RDF, OWL, SPARQL 기반 의미 데이터 표현
- 검색서비스 구조: 의미기반 검색엔진, 온톨로지
- 지능형 Agent: 자율적 정보 수집·분석·추천
- 상황인식 컴퓨팅: 센서, 컨텍스트 인식
- Big Data: 대용량 데이터 수집·저장·분석
- 클라우드 컴퓨팅: 인프라 및 서비스 제공 기반
## 구성도
```
[Web 진화 단계]

  Web 1.0        Web 2.0        Web 3.0         Web 4.0
  (읽기)    →   (읽기/쓰기)  →  (지능형)    →   (공생)
  일방향         양방향         개인맞춤형       인간-기계공생
  HTML           AJAX/SNS      Semantic Web     Symbiotic Web
                 참여/공유      개소융           All Device

[Web 3.0 구성] 개소융
  Semantic Web ── 지능형 Agent ── 상황인식 컴퓨팅
       │               │               │
   의미기반검색     Big Data       클라우드컴퓨팅
```
