#정보관리기술사 #디지털서비스 #클라우드보안 #CSAP #옵션 #정리

## 정의
클라우드 서비스 정보보호 인증 제도, CSAP
- 클라우드 서비스 제공자가 제공하는 서비스에 대해 '클라우드컴퓨팅법' 제23조 제2항에 따라 정보보호 기준의 준수 여부를 인증기관(KISA)이 평가/인증하여 이용자들이 안심하고 클라우드 서비스를 이용할 수 있도록 지원하는 제도
- CSAP: Cloud Security Assurance Program
## 키워드
* 클라우드컴퓨팅법, KISA, 최초평가, 사후평가, 갱신평가, 정보보호 기준
## 암기법
* CSAP = Cloud Security Assurance Program
* 최사갱 (최초평가 → 사후평가 → 갱신평가, 1년 단위)
## 특징
- 법적 강제성: 클라우드컴퓨팅법 제5조, 정보보호에 관한 기준 고시 제7조 근거
- 주기적 인증: 최초평가 후 매년 사후평가, 3년 주기 갱신평가
- 공공부문 필수: 공공기관 클라우드 도입 시 CSAP 인증 서비스 의무 사용
- 등급제 운영: SaaS 표준/간편, IaaS/PaaS 등급별 차등 평가
## 구성요소
| 체계 | 역할 | 설명 |
|------|------|------|
| 이용자 | 서비스 이용 | CSAP 인증 클라우드 서비스 이용 |
| 신청기관 | 인증 신청 | 클라우드 서비스 제공자(CSP) |
| 평가/인증기관 | 평가 수행 | KISA(한국인터넷진흥원) |
| 인증위원회 | 인증 심의 | 인증 적합성 최종 심의/의결 |
| 기술자문기관 | 기술 자문 | 공공부문 기술자문 수행 |
| 정책기관 | 정책 수립 | 과학기술정보통신부 |
## 구성도
```
┌────────┐  신청  ┌──────────┐  평가  ┌────────────┐
│신청기관│ ────→ │평가/인증  │ ────→ │ 인증위원회  │
│ (CSP)  │       │기관(KISA)│       │ (심의/의결) │
└────────┘       └──────────┘       └────────────┘
                       │
    ┌──────────────────┼──────────────────┐
    ▼                  ▼                  ▼
┌────────┐     ┌──────────────┐    ┌──────────┐
│이용자  │     │기술자문기관   │    │정책기관   │
│        │     │(공공부문)     │    │(과기정통부)│
└────────┘     └──────────────┘    └──────────┘

[평가주기] 최초평가 → 사후평가 → 갱신평가 (1년 단위)
```
