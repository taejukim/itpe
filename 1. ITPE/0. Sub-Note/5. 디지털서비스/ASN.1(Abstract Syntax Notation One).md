#정보관리기술사 #디지털서비스 #ASN1 #데이터명세 #옵션 #정리

## 정의
플랫폼 독립적 데이터 구조 표준 명세 언어, ASN.1(Abstract Syntax Notation One)
- 데이터 구조를 플랫폼·언어에 독립적으로 기술하기 위한 국제 표준 데이터 명세 언어
- ITU-T에서 정의한 네트워크 상의 데이터 교환을 위한 표준화된 개방형 시스템간 상호접속(OSI)의 추상적 구문 표기법
## 키워드
* ITU-T, OSI, BER, DER, PER, XER, TLV, 추상구문, 전송구문, X.680
## 암기법
* TLV (Tag, Length, Value) - ASN.1 인코딩 기본 구조
## 특징
- 플랫폼 독립성: 운영체제·프로그래밍 언어에 무관한 데이터 표현
- 추상적 표기: 데이터의 논리적 구조만을 정의하여 구현과 분리
- 다양한 인코딩: BER, DER, PER, XER 등 다양한 전송 구문 지원
- 국제 표준: ITU-T X.680 시리즈로 표준화
## 목적
- 이기종 시스템 간 데이터 교환의 호환성 보장
- 네트워크 프로토콜의 표준화된 데이터 구조 정의
## 구성요소
| 구분     | 구성요소                                                  | 설명                |
| ------ | ----------------------------------------------------- | ----------------- |
| 구문 정의  | 추상 구문(Abstract Syntax)                                | 데이터 타입·구조의 논리적 정의 |
| 인코딩    | 전송 구문(Transfer Syntax)                                | 실제 전송을 위한 바이트 인코딩 |
| 기본 타입  | INTEGER, BOOLEAN, BIT STRING 등                        | ASN.1 기본 데이터 타입   |
| 구조 타입  | SEQUENCE, SET, CHOICE                                 | 복합 데이터 구조 정의      |
| 인코딩 규칙 | BER(Basic), DER(Distinguished), PER(Packed), XER(XML) | 다양한 인코딩 방식        |
| 인코딩 구조 | TLV (Tag-Length-Value)                                | 태그-길이-값 기반 인코딩 포맷 |
## 인코딩 예시
```
[ASN.1 정의]
  age INTEGER ::= 5

[BER 인코딩 결과] (TLV 구조)
  02  01  05
  │   │   └── Value: 5 (값)
  │   └────── Length: 1바이트 (길이)
  └────────── Tag: INTEGER 타입 (태그)

[인코딩 규칙 비교]
  BER: 기본 인코딩 규칙 (유연, 다양한 표현)
  DER: 정규화 인코딩 (인증서 등 단일 표현 필요 시)
  PER: 압축 인코딩 (통신 효율 극대화)
  XER: XML 기반 인코딩 (가독성 확보)
```
## 구성도
```
[ASN.1 동작 구조]
  ┌─────────────┐         ┌─────────────┐
  │  시스템 A    │         │  시스템 B    │
  │  (송신)      │         │  (수신)      │
  │             │         │             │
  │ 추상 구문   │         │ 추상 구문   │
  │ (데이터정의) │         │ (데이터정의) │
  │      │      │         │      ▲      │
  │ 인코딩(BER) │ ──────▶ │ 디코딩(BER) │
  │ (전송구문)  │  네트워크 │ (전송구문)  │
  └─────────────┘         └─────────────┘

[적용 분야]
  X.509 인증서, SNMP, LDAP, 3GPP 이동통신
```
