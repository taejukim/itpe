#정보관리기술사 #디지털서비스 #ADAS #자율주행 #옵션

## 정의
센서 기반 운전자 보조 시스템, ADAS(Advanced Driver Assistance System)
- 센서, 카메라, 레이더, 초음파 등을 활용하여 주행 환경을 인지하고, 운전자의 안전 운행을 보조하는 첨단 운전자 지원 시스템
- 인지·판단·제어 기반으로 차선유지, 충돌경보, 자동제동 등 운전 보조 기능을 제공
## 키워드
* 인지-판단-제어, LDW, LKA, SCC, ACC, BSD, FCWS, AEB, ERTICO, ADASIS, NDS, REM
## 암기법
* 인판제 (인지, 판단, 제어) - ADAS 3대 핵심 구성
## 특징
- 차선유지 기술: 차선이탈경보(LDW) 및 차선유지지원(LKA) 제공
- 주행 보조 기술: 스마트크루즈컨트롤(SCC), 어댑티브크루즈컨트롤(ACC)
- 충돌 예방 기술: 후측방경보(BSD), 전방추돌경보(FCWS) 제공
- 자동 제동 기술: 자동긴급제동(AEB)으로 충돌 방지
## 목적
- 교통사고 예방 및 운전자 안전성 향상
- 자율주행 기술로의 단계적 전환 기반 마련
## 구성요소
| 구분 | 기술 | 설명 |
|------|------|------|
| 인지 | 센서, 카메라, 레이더, 초음파 | 주행 환경 인식 및 객체 감지 |
| 판단 | AI/ECU 알고리즘 | 위험 상황 판단 및 대응 결정 |
| 제어 | 조향/가감속/제동 | 차량 제어 명령 실행 |
| 차선유지 | LDW(차선이탈경보), LKA(차선유지지원) | 차선 이탈 방지 |
| 주행보조 | SCC(스마트크루즈), ACC(어댑티브크루즈) | 속도·거리 자동 유지 |
| 충돌경보 | BSD(후측방경보), FCWS(전방추돌경보) | 충돌 위험 사전 경고 |
| 제동 | AEB(자동긴급제동) | 긴급 상황 자동 제동 |
| 표준 | ERTICO(ADASIS, NDS) | 도로 데이터 표준화 |
| 데이터 | REM(Road Experience Management) | 도로 경험 데이터 관리 |
## 구성도
```
[ADAS 아키텍처] (인판제)
  ┌─────────┐    ┌─────────┐    ┌─────────┐
  │  인지    │ →  │  판단    │ →  │  제어    │
  │ 센서     │    │ AI/ECU  │    │ 조향     │
  │ 카메라   │    │ 알고리즘 │    │ 가감속   │
  │ 레이더   │    │         │    │ 제동     │
  │ 초음파   │    │         │    │         │
  └─────────┘    └─────────┘    └─────────┘

[주요 기술 분류]
  차선유지: LDW(이탈경보) ── LKA(유지지원)
  주행보조: SCC(스마트크루즈) ── ACC(어댑티브크루즈)
  충돌경보: BSD(후측방) ── FCWS(전방추돌)
  자동제동: AEB(긴급제동)

[표준] ERTICO → ADASIS(도로데이터) + NDS(네비DB)
[국내] 2018년 대형차 ADAS 의무 장착
```
