#정보관리기술사 #디지털서비스 #메타버스 #가상세계

## 정의
**다감각 3차원 가상 소통 세계, 메타버스(Metaverse)
- 다감각 3차원 환경에서 사용자가 아바타를 통해 의사소통하고 정화/재화를 교환하면서 현실세계와 소통하는 세계
- 현실과 가상의 경계를 초월하여 경제·사회·문화 활동이 이루어지는 3차원 가상 융합 공간
## 키워드
* 아바타, AR, VR, 라이프로깅, 미러월드, 가상세계, NFT, 디지털트윈, XR
## 암기법
* 증라미가 (증강현실, 라이프로깅, 미러월드, 가상세계) - 4대 범주
## 특징
- 증강현실(AR): 현실 위에 가상 정보를 중첩하는 Augmentation 기반 기술
- 라이프로깅: 일상을 디지털로 기록·공유하는 Intimate(내적) 기반 서비스
- 미러월드: 현실세계를 가상에 그대로 복제한 External(외적) Simulation 공간
- 가상세계(VW): 현실과 다른 완전한 가상 환경에서 활동하는 디지털 세계
## 목적
- 현실세계의 물리적 한계를 초월한 경제·사회·문화 활동 공간 제공
- 몰입형 사용자 경험 기반의 새로운 비즈니스 모델 창출
## 구성요소
- 증강현실(AR): Augmentation + External, 현실 기반 가상 정보 증강
- 라이프로깅: Augmentation + Intimate, 일상 기록 및 공유
- 미러월드: Simulation + External, 현실세계의 가상 복제
- 가상세계(VW): Simulation + Intimate, 완전한 가상 환경
- 아바타: 가상세계 내 사용자 대리 캐릭터
- 디지털 자산: NFT, 가상 화폐 기반 경제 시스템
## 서비스
- 가상홈: 가상 공간 내 개인 생활 공간
- 가상오피스: 원격 협업 가상 업무 환경
- 포털서비스: 메타버스 진입 및 서비스 허브
- E-헬스: 가상 환경 기반 건강관리 서비스
- 인스턴트 콘텐츠/전자상거래: 실시간 콘텐츠 생성 및 거래
- E-러닝: 몰입형 가상 교육 서비스
- 가상커뮤니티: 사용자 간 소통·교류 공간
## 구성도

```
          Augmentation(증강)

    ┌──────────┼──────────┐
    │          │          │
External ── 증강현실(AR) │   Intimate
(외적)      │          │    (내적)
    │     미러월드    라이프로깅
    │          │          │
    └──────────┼──────────┘
               │
          Simulation(시뮬레이션)
            가상세계(VW)

[서비스]
  가상홈 ─ 가상오피스 ─ 포털서비스
  E-헬스 ─ E-러닝 ─ 가상커뮤니티
  인스턴트 콘텐츠 ─ 전자상거래
```
