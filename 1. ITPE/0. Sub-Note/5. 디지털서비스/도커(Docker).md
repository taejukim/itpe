## 정의
컨테이너 기반 경량 가상화 플랫폼, Docker
- LXC(Linux Container) 기반의 컨테이너 기술을 활용하여 애플리케이션과 실행 환경을 패키징하고 배포하는 오픈소스 가상화 플랫폼
- VMM, 하이퍼바이저 대비 OS 레벨 가상화로 경량화된 실행 환경 제공
## 키워드
* DockerFile, Docker Image, Docker Container, Docker Hub, LXC, 레이어, 이미지 레지스트리
## 암기법
* 파이컨허 (DockerFile → Docker Image → Docker Container → Docker Hub)
## 특징
- 경량화: Guest OS 없이 프로세스 격리 방식으로 하이퍼바이저 대비 오버헤드 최소화
- 이식성: 이미지 기반 패키징으로 어떤 환경에서든 동일하게 실행
- 빠른 시작: 컨테이너 기동 시간이 초 단위로 VM 대비 매우 빠름
- 레이어 구조: 이미지의 레이어 캐싱을 통한 효율적 저장 및 배포
## 구성요소
| 구성요소 | 역할 | 설명 |
|----------|------|------|
| DockerFile | 이미지 정의 | 이미지 빌드를 위한 명세 파일 |
| Docker Image | 실행 템플릿 | 컨테이너 실행을 위한 읽기 전용 템플릿 |
| Docker Container | 실행 인스턴스 | 이미지를 기반으로 생성된 격리된 실행 환경 |
| Docker Hub | 이미지 저장소 | 이미지를 공유/배포하는 레지스트리 |
| Docker Engine | 실행 엔진 | 컨테이너의 생성, 실행, 관리를 담당 |
