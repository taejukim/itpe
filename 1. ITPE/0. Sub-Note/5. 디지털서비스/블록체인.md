## 정의
분산원장 기반 신뢰 보장 기술, 블록체인
- P2P 네트워크에서 거래 데이터를 블록 단위로 기록하고, 암호화 해시로 연결하여 위·변조가 불가능한 분산원장기술(DLT)
- 중앙기관 없이 참여자 간 합의 알고리즘을 통해 거래의 신뢰성과 투명성을 보장하는 탈중앙화 기술
## 키워드
* DLT, Mining, PoW, TPS, 스마트 계약, 하이퍼레저, R3, BaaS, 개방형/폐쇄형, 멤블체
## 암기법
* 멤블체 - 멤버쉽 서비스, 블록체인 서비스, 체인코드 서비스 (핵심기술)
* 페사이인버 - 하이퍼레저 Framework (Fabric, Sawtooth, Iroha, INDY, Burrow)
* 컴칼셀익퀄 - 하이퍼레저 TOOL (Composer, CALIPER, Cello, Explorer, QUILT)
## 특징
- 탈중앙화: 중앙기관 없이 P2P 네트워크로 분산 운영
- 투명성: 모든 참여자가 동일한 원장 데이터를 공유·검증
- 불변성: 해시 체인 구조로 한번 기록된 데이터 위·변조 불가
- 합의 기반: PoW, PoS 등 합의 알고리즘을 통한 거래 검증
## 목적
- 중앙기관 의존 없는 거래 신뢰성 확보 및 비용 절감
- 데이터 위·변조 방지를 통한 투명하고 안전한 거래 환경 구축
## 구성요소
| 구분 | 구성요소 | 설명 |
|------|----------|------|
| 분류 | 개방형(Public) | 누구나 참여 가능, 비트코인·이더리움 |
| 분류 | 폐쇄형(Private) | 허가된 참여자만 접근, R3·하이퍼레저 |
| 핵심기술 | 멤버쉽 서비스 | 외부등록, 인증, 감사 |
| 핵심기술 | 블록체인 서비스 | Http/2, P2P, 해시 알고리즘 |
| 핵심기술 | 체인코드 서비스 | 보안 OS, Go lang, Java, Node.js |
| 마이닝 | Proof-of-Work | 작업증명 기반 채굴(Mining) |
| 성능 | TPS | 초당 거래 처리건수(Transaction Per Second) |
| 서비스 | BaaS | BPaaS(개발플랫폼), BSaaS(SW제공) |
| 계약 | 스마트 계약 | 제3자 개입 없이 자동 계약 실행 |
