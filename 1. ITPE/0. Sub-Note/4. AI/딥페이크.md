#AI #보안 #생성AI #필수

## 정의
딥러닝 기반 영상 합성 기술, 딥페이크
- 딥러닝을 이용해 기존 영상에 다른 영상이나 이미지 정보를 합성하여 콘텐츠를 생성하는 기법
- Deep Learning + Fake의 합성어
## 키워드
* Deepfake, GAN, Autoencoder, Face Swap, 탐지, 합성
## 암기법
* Deepfake = Deep(딥러닝) + Fake(가짜)
* 생성: GAN/AE → 탐지: CNN
## 연관 토픽
- [[GAN (Generative Adversarial Network)]] - 생성 모델
- [[오토인코더(Autoencoder)]] - 인코더-디코더
- [[AI 윤리]] - 윤리적 문제
## 생성 과정
```
┌─────────────────────────────────────────────────┐
│              딥페이크 생성 파이프라인                 │
│                                                 │
│  1. 데이터 수집 → 2. 얼굴 인식 → 3. 특징 추출          │
│         │              │              │         │
│         ▼              ▼              ▼         │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐     │
│  │ 원본영상   │   │ 얼굴탐지    │   │ 인코딩    │     │
│  │ 타겟영상   │   │ 랜드마크    │   │ 임베딩    │     │
│  └──────────┘   └──────────┘   └──────────┘     │ 
│                                                 │
│  4. 생성모델 → 5. 합성 → 6. 후처리                   │
│         │          │          │                 │
│         ▼          ▼          ▼                 │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐     │
│  │ GAN/AE   │   │ 얼굴교체   │   │  블렌딩    │     │
│  │ 학습      │   │ 합성      │   │ 색보정     │     │
│  └──────────┘   └──────────┘   └──────────┘     │
└─────────────────────────────────────────────────┘
```
## 핵심 기술
| 단계 | 기술 |
|------|------|
| 얼굴 인식 | Face Detection, Landmark Detection |
| 특징 추출 | Feature Extraction, Embedding |
| 생성 모델 | ==GAN 기반== 또는 ==Autoencoder 기반== |
| 합성 | Face Swapping, Reenactment |
| 후처리 | Blending, Color Correction |
## 딥페이크 유형
| 유형 | 설명 |
|------|------|
| Face Swap | 얼굴 교체 |
| Face Reenactment | 표정/동작 전이 |
| Lip Sync | 입술 동기화 |
| Voice Deepfake | 음성 합성 |
| Full Body | 전신 합성 |
## 탐지 기술
- ==생물학적 신호==: 눈 깜빡임, 맥박 패턴
- ==물리적 불일치==: 조명, 그림자, 반사
- ==압축 아티팩트==: 영상 압축 흔적
- ==딥러닝 탐지==: CNN 기반 진위 판별
- ==주파수 분석==: 고주파 성분 분석
