#AI #딥러닝 #최적화 #필수

## 정의
모델의 최적 파라미터를 찾는 알고리즘, 최적화 알고리즘
- AI·머신러닝 모델이 가장 좋은(최적) 파라미터 값을 찾기 위해 사용하는 알고리즘
- 손실 함수를 최소화하는 방향으로 가중치를 업데이트
## 키워드
* Optimization, SGD, Adam, AdamW, 학습률, Momentum
## 암기법
* 최적화 발전: ==SGD → Momentum → Adam==
* Adam = Adaptive(적응) + Moment(모멘트) Estimation

- [[과적합(Overfitting) 문제]] - 일반화

## 주요 최적화 알고리즘
| 알고리즘                                      | 특징                  | 장단점       |
| ----------------------------------------- | ------------------- | --------- |
| ==SGD==                                   | 확률적 경사하강법           | 단순, 느린 수렴 |
| ==Momentum==                              | 이전 기울기 반영           | 진동 감소     |
| ==RMSProp==(Root Mean Square Propagation) | 학습률 자동 조정           | 적응적 학습률   |
| ==Adam==                                  | Momentum + RMSProp  | 가장 대중적    |
| ==AdamW==                                 | Adam + Weight Decay | 대규모 모델 기본 |
- 네스테로프 모멘텀
- AdaGrad
## Adam 알고리즘
```
m_t = β₁ × m_{t-1} + (1-β₁) × g_t     (1차 모멘트)
v_t = β₂ × v_{t-1} + (1-β₂) × g_t²    (2차 모멘트)

m̂_t = m_t / (1-β₁^t)                  (편향 보정)
v̂_t = v_t / (1-β₂^t)

θ_t = θ_{t-1} - α × m̂_t / (√v̂_t + ε)  (파라미터 업데이트)
```
## 최적화 알고리즘 발전
```
┌─────────────────────────────────────────────────┐
│              최적화 알고리즘 발전                    │
│                                                 │
│  SGD → Momentum → NAG → AdaGrad → RMSProp → Adam│
│   │        │       │        │         │       │ │
│   │        │       │        │         │       │ │
│   ▼        ▼       ▼        ▼         ▼       ▼ │
│ 기본    관성추가     예측     적응학습률    개선    통합  │
└─────────────────────────────────────────────────┘
```
## 학습률 스케줄링
| 방법 | 설명 |
|------|------|
| Step Decay | 일정 에폭마다 감소 |
| Exponential Decay | 지수적 감소 |
| Cosine Annealing | 코사인 함수 기반 |
| Warmup | 초기에 점진적 증가 |
| ReduceLROnPlateau | 성능 정체 시 감소 |
## 하이퍼파라미터
| 파라미터 | 설명 | 일반적 값 |
|----------|------|-----------|
| 학습률 (α) | 업데이트 크기 | 0.001 |
| β₁ | 1차 모멘트 감쇠율 | 0.9 |
| β₂ | 2차 모멘트 감쇠율 | 0.999 |
| ε | 수치 안정성 | 1e-8 |
