#AI #신뢰성 #품질 #필수

## 정의
AI 시스템의 신뢰할 수 있는 품질 수준, 인공지능 신뢰성
- 인공지능(AI) 모델이 학습한 결과에 대해서 주어진 문제를 얼마나 정확하고 신뢰성 있게 해결하는지를 수치화하여 판단하는 과정
- 안전하고 공정하며 설명 가능한 AI 구현
## 키워드
* Trustworthiness, XAI, SHAP, LIME, 공정성, 설명가능성
## 암기법
* 신뢰성 요소: ==정공설안프견== (정확성, 공정성, 설명가능성, 안전성, 프라이버시, 견고성)

## 평가 지표
| 영역 | 지표 |
|------|------|
| 정확성 | 혼동행렬, F1 Score, ROC AUC, MAE, MSE, RMSE |
| 공정성 | 편향성 검증, 그룹 간 성능 차이 |
| 견고성 | 적대적 공격 저항성, 노이즈 내성 |
| 설명가능성 | SHAP, LIME, Attention 시각화 |
| 일반화 | 과적합/과소적합 판단 |
## 신뢰성 확보 방안
| 단계 | 방안 |
|------|------|
| 데이터 | 품질 관리, 편향 제거, 다양성 확보 |
| 모델 | 검증, 테스트, 설명가능성 확보 |
| 운영 | 모니터링, 감사, 피드백 반영 |
| 거버넌스 | 정책, 프로세스, 조직 체계 |
## 설명가능 AI (XAI)
- ==SHAP==: 특성 기여도 분석
- ==LIME==: 로컬 해석 가능 모델
- ==Attention==: 어텐션 가중치 시각화
- ==Feature Importance==: 특성 중요도
