#AI #인공지능 #미래기술 #필수

## 정의
인간 수준의 범용 지능을 갖춘 인공지능, AGI
- 다양한 분야에서 인간과 유사한 학습, 추론, 문제해결 능력을 보유한 인공지능
- 특정 작업에 한정되지 않고 범용적으로 적용 가능한 지능 시스템
## 키워드
* 범용 인공지능, 강인공지능, 튜링 테스트, 자기인식, 위험관리
## 암기법
* AGI = Artificial(인공) + General(범용) + Intelligence(지능)

## ANI vs AGI vs ASI
| 구분  | ANI (약인공지능)  | AGI (강인공지능) | ASI (초인공지능) |
| --- | ------------ | ----------- | ----------- |
| 범위  | 특정 작업        | 범용 작업       | 인간 초월       |
| 예시  | 알파고, ChatGPT | 미래 목표       | SF 영역       |
| 현재  | 상용화          | 연구 중        | 이론적         |
|     |              |             |             |
## 핵심 특성
- ==범용성(Generality)==: 다양한 도메인에서 작동
- ==전이 학습(Transfer Learning)==: 한 분야 지식을 다른 분야에 적용
- ==추론 능력(Reasoning)==: 논리적 사고와 문제 해결
- ==자기 학습(Self-Learning)==: 스스로 학습하고 개선
- ==상식 추론(Common Sense)==: 일반적 지식 기반 판단
## 위험관리 프레임워크
- 위험 식별, 분석, 평가, 대응
- 위험 프로필 작성
- 3D 위험 매트릭스 활용
- 기술적, 윤리적, 사회적, 법적 위험 요소 관리
