#AI #머신러닝 #자동화

## 정의
머신러닝 파이프라인을 자동화하는 기술, AutoML
- 데이터 전처리, 특성 엔지니어링, 모델 선택, 하이퍼파라미터 튜닝 등 ML 파이프라인 전 과정을 자동화하는 기술
- 비전문가도 ML 모델을 쉽게 구축할 수 있게 지원
## 키워드
* AutoML, NAS, HPO, 하이퍼파라미터, 자동화, 모델 선택
## 암기법
* AutoML = Automated(자동화) + ML
* 자동화 영역: ==전특모튜== (전처리, 특성, 모델, 튜닝)
* NHFM : NAS, HPO, FE, MS 
## 연관 토픽
- [[경사하강법]] - 최적화 알고리즘
- [[과적합(Overfitting) 문제]] - 모델 일반화
- [[인공지능 파운데이션 모델]] - 기반 모델
## 자동화 영역
```
┌─────────────────────────────────────────────────┐
│              AutoML 파이프라인                     │
│                                                 │
│  [데이터]→[전처리]→[특성]→[모델]→[튜닝]→[배포]          │
│           ↓        ↓      ↓      ↓              │
│         자동화   자동화  자동화  자동화                │
│                                                 │
│  - 결측치 처리    - 특성 선택  - 모델 선택             │
│  - 정규화         - 특성 생성  - 앙상블               │
│  - 인코딩         - 차원 축소  - 하이퍼파라미터         │
└─────────────────────────────────────────────────┘
```
## 핵심 기술
| 기술 | 설명 |
|------|------|
| ==NAS (Neural Architecture Search)== | 신경망 구조 자동 탐색 |
| ==HPO (Hyperparameter Optimization)== | 하이퍼파라미터 최적화 |
| ==Feature Engineering== | 자동 특성 생성/선택 |
| ==Model Selection== | 최적 모델 자동 선택 |
## 하이퍼파라미터 최적화 방법
| 방법 | 설명 |
|------|------|
| Grid Search | 격자 탐색 (모든 조합) |
| Random Search | 무작위 탐색 |
| Bayesian Optimization | 베이지안 최적화 |
| Genetic Algorithm | 유전 알고리즘 |
