#AI #데이터 #품질관리

## 정의
AI 학습 데이터의 품질을 체계적으로 관리하는 활동, 데이터 품질관리
- AI 모델의 성능과 신뢰성을 보장하기 위해 학습용 데이터의 수집, 가공, 검증 전 과정에서 품질을 관리하는 체계
- =="Garbage In, Garbage Out"== 방지를 위한 핵심 활동
## 키워드
* 데이터 품질, 라벨링, 정확성, 완전성, 일관성, 대표성
## 암기법
* 품질 요소: ==정완일적유대== (정확성, 완전성, 일관성, 적시성, 유효성, 대표성)
* 프로세스: ==수정라검관== (수집, 정제, 라벨링, 검증, 관리)

## 데이터 품질 요소
| 요소 | 설명 |
|------|------|
| ==정확성(Accuracy)== | 데이터가 실제 값과 일치 |
| ==완전성(Completeness)== | 필요한 데이터가 모두 존재 |
| ==일관성(Consistency)== | 데이터 간 모순 없음 |
| ==적시성(Timeliness)== | 최신 데이터 유지 |
| ==유효성(Validity)== | 정의된 규칙 준수 |
| ==대표성(Representativeness)== | 모집단을 대표 |
## 품질관리 프로세스
```
┌─────────────────────────────────────────────────┐
│           데이터 품질관리 프로세스                    │
│                                                 │
│  [수집] → [정제] → [라벨링] → [검증] → [관리]         │
│    │       │       │        │        │          │
│    ▼       ▼       ▼        ▼        ▼          │
│  출처확인  결측치    품질기준   교차검증    버전         │
│  동의확보  이상치    가이드    샘플검증     관리         │
│  다양성    정규화    일관성    전문가     모니터        │
└─────────────────────────────────────────────────┘
```
## 라벨링 품질관리
| 항목 | 내용 |
|------|------|
| ==라벨링 가이드== | 명확한 기준 및 예시 제공 |
| ==작업자 교육== | 라벨링 품질 교육 |
| ==교차 검증== | 다수 작업자 결과 비교 |
| ==전문가 검토== | 샘플 검증 및 피드백 |
| ==품질 지표== | 일치율, 정확도 측정 |
## 품질 검증 방법
| 방법 | 설명 |
|------|------|
| 통계적 검증 | 분포, 이상치, 결측치 분석 |
| 샘플링 검증 | 무작위 샘플 품질 확인 |
| 교차 검증 | 작업자 간 일치도 |
| 자동화 검증 | 규칙 기반 자동 검사 |
## 품질 지표
| 지표 | 설명 |
|------|------|
| ==Inter-Annotator Agreement== | 작업자 간 일치도 |
| ==Cohen's Kappa== | 우연 일치 보정 일치도 |
| ==라벨 정확도== | 정답 대비 정확률 |
| ==완성률== | 라벨링 완료 비율 |
