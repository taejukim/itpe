#AI #로봇 #자율주행

## 정의
물리적 세계와 상호작용하는 인공지능, 피지컬 AI
- 로봇, 자율주행차와 같은 물리적 기기에 탑재되어, 물리적 세계를 인식하고 이해하며 상호작용하는 인공지능
- 디지털 세계를 넘어 실제 물리 환경에서 동작하는 AI
## 키워드
* Physical AI, 휴머노이드, 자율주행, LWM, 디지털트윈, 온디바이스
## 암기법
* 피지컬 AI = Physical(물리적) + AI
* 인식 → 이해 → 행동 (Sense → Think → Act)

## 핵심 기술
| 기술 | 설명 |
|------|------|
| ==LWM(Large World Model)== | 물리 세계 이해 모델 |
| ==디지털 트윈== | 물리 환경 시뮬레이션 |
| ==온디바이스 AI== | 실시간 로컬 처리 |
| ==모델 경량화== | sLLM, 양자화 |
| ==벡터 DB== | 환경 정보 저장/검색 |
| ==강화학습== | 행동 최적화 |
## 적용 분야
| 분야 | 예시 |
|------|------|
| ==휴머노이드== | 테슬라 옵티머스, 보스턴다이나믹스 |
| ==자율주행== | 웨이모, 테슬라 FSD |
| ==산업용 로봇== | 제조, 물류 자동화 |
| ==드론== | 배송, 검사, 농업 |
| ==의료 로봇== | 수술 보조, 재활 |
## 기술 스택
- ==인식==: 컴퓨터 비전, LiDAR, 센서 퓨전
- ==이해==: LLM, 상황 인식, 의도 파악
- ==계획==: 경로 계획, 작업 계획
- ==제어==: 모터 제어, 매니퓰레이션
- ==학습==: 강화학습, 모방 학습
