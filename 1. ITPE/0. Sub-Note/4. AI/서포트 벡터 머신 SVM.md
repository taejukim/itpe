#AI #머신러닝 #분류

## 정의
최적의 결정 경계를 찾는 지도학습 분류 알고리즘, SVM
- 기계 학습의 분야 중 하나로 패턴 인식, 자료 분석을 위한 지도 학습 모델
- 두 카테고리 중 어느 하나에 속한 데이터의 집합이 주어졌을 때, 새로운 데이터가 어느 카테고리에 속할지 판단하는 비확률적 이진 선형 분류 모델
## 키워드
* Support Vector, Margin, Hyperplane, Kernel, 분류, 회귀분석
## 암기법
* SVM = Support(지지) + Vector(벡터) + Machine(기계)
* 마진 최대화 = 일반화 성능 최대화
## 연관 토픽
- [[KNN]] - K-최근접 이웃
- [[CNN]] - 합성곱 신경망
- [[과적합(Overfitting) 문제]] - 일반화 문제
## 핵심 개념
- ==서포트 벡터(Support Vector)==: 결정 경계에 가장 가까운 데이터 포인트
- ==마진(Margin)==: 서포트 벡터와 결정 경계 사이의 거리
- ==결정 경계(Decision Boundary)==: 클래스를 분리하는 초평면(Hyperplane)
- ==커널(Kernel)==: 비선형 분류를 위한 차원 변환 함수
## 구조
```
        │
    ○   │   ●
  ○     │     ●
    ○ ──┼── ●    ← 결정 경계 (Hyperplane)
  ○     │     ●
    ○   │   ●
        │
   ◀────┼────▶
      Margin
```
## 커널 종류
| 커널 | 특징 | 활용 |
|------|------|------|
| 선형(Linear) | 선형 분리 가능 데이터 | 텍스트 분류 |
| 다항식(Polynomial) | 비선형 경계 | 이미지 처리 |
| RBF(Gaussian) | 복잡한 비선형 | 범용적 사용 |
| 시그모이드(Sigmoid) | 신경망 유사 | 특수 케이스 |
## 장단점
| 장점 | 단점 |
|------|------|
| 고차원 데이터에 효과적 | 대용량 데이터 학습 느림 |
| 과적합에 강함 | 커널/파라미터 선택 어려움 |
| 메모리 효율적 | 다중 클래스 분류 복잡 |
## 활용 분야
- 텍스트 분류 (스팸 필터링)
- 이미지 인식
- 생물정보학 (유전자 분류)
- 필기 인식
