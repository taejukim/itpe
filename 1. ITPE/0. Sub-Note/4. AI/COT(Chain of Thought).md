#AI #LLM #추론

## 정의
단계별 추론을 유도하는 프롬프팅 기법, COT
- LLM이 복잡한 문제를 해결할 때 중간 추론 단계를 명시적으로 생성하도록 유도하는 프롬프팅 기법
- "Let's think step by step"으로 단계별 사고 과정 유도
## 키워드
* Chain of Thought, 단계별 추론, 프롬프팅, Zero-shot, Few-shot
## 암기법
* COT = Chain(사슬) + of + Thought(생각)
* "Let's think step by step"
## 연관 토픽
- [[LLM]] - 대규모 언어 모델
- [[RAG(Retrieval Augmented Generation)]] - 검색 증강 생성
- [[MOE(Mixture of Experts)]] - 전문가 혼합
## 동작 원리
```
┌─────────────────────────────────────────────────┐
│              일반 프롬프팅                         │
│  Q: 8+5×3=?                                     │
│  A: 23                                          │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│              COT 프롬프팅                         │
│  Q: 8+5×3=?                                     │
│  A: 먼저 곱셈을 계산합니다: 5×3=15                   │
│     그 다음 덧셈을 계산합니다: 8+15=23                │
│     따라서 답은 23입니다.                           │
└─────────────────────────────────────────────────┘
```
## COT 유형
| 유형                   | 설명                            | 특징     |
| -------------------- | ----------------------------- | ------ |
| ==Zero-shot COT==    | "Let's think step by step" 추가 | 예시 불필요 |
| ==Few-shot COT==     | 추론 예시 제공                      | 더 정확함  |
| ==Self-Consistency== | 여러 추론 경로 생성 후 투표              | 신뢰도 향상 |
| ==Tree of Thoughts== | 트리 구조 탐색                      | 복잡한 문제 |
## 효과
| 영역 | 개선 효과 |
|------|-----------|
| 수학 문제 | 정확도 대폭 향상 |
| 논리 추론 | 복잡한 추론 가능 |
| 상식 추론 | 단계별 분석 |
| 코드 생성 | 로직 설명 포함 |
## 프롬프트 예시
```
Q: 철수는 사과 5개를 가지고 있었습니다. 
   영희에게 2개를 주고, 3개를 더 샀습니다.
   철수는 몇 개의 사과를 가지고 있나요?

A: 단계별로 생각해봅시다.
   1. 처음 철수가 가진 사과: 5개
   2. 영희에게 준 후: 5 - 2 = 3개
   3. 더 산 후: 3 + 3 = 6개
   따라서 철수는 6개의 사과를 가지고 있습니다.
```
## 관련 기법
- ==Self-Consistency==: 다중 추론 경로 + 다수결
- ==Tree of Thoughts (ToT)==: 트리 구조 탐색
- ==Graph of Thoughts (GoT)==: 그래프 구조 추론
- ==ReAct==: 추론 + 행동 결합
