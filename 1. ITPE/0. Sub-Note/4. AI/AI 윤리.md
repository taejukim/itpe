## 정의
AI 개발과 활용에서의 윤리적 원칙과 가이드라인, AI 윤리
- 인공지능 기술의 개발, 배포, 활용 전 과정에서 준수해야 할 도덕적 원칙과 사회적 책임
- 인간 중심의 AI 구현을 위한 윤리적 프레임워크
## 키워드
* 공정성, 투명성, 책임성, 편향, 프라이버시, 인간 중심
## 암기법
- AI 3대 원칙 : ==인사기== (인간존엄성 원식, 사회 공공성 기술 합목적성)
* AI 핵심 요건: ==투안책데 연공침다 프인== (투명성, 안정성, 책임성, 데이터관리, 연대성, 공공성, 침해 금지, 다양성, 프라이버시, 인간존중)

## 핵심 원칙
| 원칙       | 설명                                  |
| -------- | ----------------------------------- |
| 투명성      | 설명 가능성, 활용 내용 및 유의 사항 가전 고지         |
| 안정성      | 잠재적 위험 방지, 안전 보장                    |
| 책임성      | 책임의 명확화, 주체별 책임                     |
| 데이터 관리   | 목적 외 용도 활용 금지, 데이터 편향 최소화, 품질 위험 관리 |
| 연대성      | 집단간 연대성, 이해관계자의 참여 기회 보장, 국제 사회 협력  |
| 공공성      | 공공성 증진, 인류의 공동 이익 목적, 순기능 극대화 교육    |
| 침해금지     | 침해금지, 인간해 무해한 목적으로 사용               |
| 다양성 존중   | 다양성, 접근성 보장, 비차별성, 편향, 차별 최소화       |
| 프라이버시 보호 | 사생활 보로, 개인정보 오용 최소화                 |
| 인권보장     | 인간 중신, 인간의 권리와 자유 보장, 사람중심 서비스      |

## 편향(Bias) 문제
| 편향 유형 | 설명 | 예시 |
|-----------|------|------|
| 데이터 편향 | 학습 데이터의 불균형 | 특정 인종/성별 과소 대표 |
| 알고리즘 편향 | 모델 설계상 편향 | 특정 그룹 불리한 결과 |
| 확증 편향 | 기존 편견 강화 | 추천 알고리즘 필터버블 |
