#AI #머신러닝 #평가

## 정의
분류 모델의 성능을 평가하는 행렬, 혼동 행렬
- 분류 모델의 예측 결과와 실제 값을 비교하여 정리한 2×2(이진 분류) 또는 n×n(다중 분류) 행렬
- True/False와 Positive/Negative의 조합으로 모델 성능 분석
## 키워드
* Confusion Matrix, TP, TN, FP, FN, Precision, Recall, F1 Score
## 암기법
* 혼동 행렬: ==TPTNFPFN==
* 평가 지표: ==정정재에프== (정확도, 정밀도, 재현율, F1)
## 구성요소
| 요소 | 설명 | 예시 (암 진단) |
|------|------|----------------|
| ==TP (True Positive)== | 양성을 양성으로 예측 | 암 환자를 암으로 진단 |
| ==TN (True Negative)== | 음성을 음성으로 예측 | 정상인을 정상으로 진단 |
| ==FP (False Positive)== | 음성을 양성으로 예측 | 정상인을 암으로 오진 (1종 오류) |
| ==FN (False Negative)== | 양성을 음성으로 예측 | 암 환자를 정상으로 오진 (2종 오류) |

## 혼동 행렬 구조 (이진 분류)
```
                    실제값
                 Positive  Negative
              ┌──────────┬──────────┐
    Positive  │    TP    │    FP    │
예측값          │ (True    │ (False   │
              │ Positive)│ Positive)│
              ├──────────┼──────────┤
    Negative  │    FN    │    TN    │
              │ (False   │ (True    │
              │ Negative)│ Negative)│
              └──────────┴──────────┘
```
## 평가 지표
| 지표 | 수식 | 설명 |
|------|------|------|
| ==정확도(Accuracy)== | (TP+TN)/(TP+TN+FP+FN) | 전체 정확률 |
| ==정밀도(Precision)== | TP/(TP+FP) | 양성 예측 중 실제 양성 |
| ==재현율(Recall)== | TP/(TP+FN) | 실제 양성 중 양성 예측 |
| ==F1 Score== | 2×(P×R)/(P+R) | 정밀도와 재현율의 조화평균 |
| ==특이도(Specificity)== | TN/(TN+FP) | 실제 음성 중 음성 예측 |
## 정밀도 vs 재현율
| 상황 | 중요 지표 | 예시 |
|------|-----------|------|
| FP 비용 높음 | 정밀도 | 스팸 필터 (정상 메일 차단 방지) |
| FN 비용 높음 | 재현율 | 암 진단 (암 환자 놓치면 안됨) |
| 균형 필요 | F1 Score | 일반적인 분류 문제 |
## ROC-AUC
- ==ROC 곡선==(Receiver Operating Characteritics): FPR vs TPR 그래프
- ==AUC==(Area Under Curve): ROC 곡선 아래 면적 (1에 가까울수록 좋음)
- 임계값에 따른 성능 변화 시각화
