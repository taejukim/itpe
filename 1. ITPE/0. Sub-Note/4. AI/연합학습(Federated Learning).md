#AI #프라이버시 #분산학습

## 정의
데이터를 공유하지 않고 분산 학습하는 기법, 연합학습
- 민감정보 보호를 위한 분산형 AI 학습 방식
- 데이터는 로컬에 유지하고 모델 파라미터만 중앙 서버와 공유하여 학습
## 키워드
* Federated, 분산학습, 프라이버시, FedAvg, 로컬 학습
## 암기법
* 연합학습 = 데이터는 로컬, 모델만 공유
* FL = Federated(연합) + Learning(학습)
## 학습 과정
1. ==초기화==: 중앙 서버에서 글로벌 모델 배포
2. ==로컬 학습==: 각 디바이스에서 로컬 데이터로 학습
3. ==모델 업로드==: 학습된 모델 파라미터(가중치) 전송
4. ==집계(Aggregation)==: 서버에서 파라미터 평균화 (FedAvg)
5. ==배포==: 업데이트된 글로벌 모델 재배포
6. ==반복==: 수렴할 때까지 2-5 반복

## 유형
| 유형 | 설명 | 예시 |
|------|------|------|
| 수평적 연합학습 | 같은 특성, 다른 샘플 | 여러 병원의 환자 데이터 |
| 수직적 연합학습 | 같은 샘플, 다른 특성 | 은행 + 보험사 데이터 |
| 연합 전이학습 | 다른 샘플, 다른 특성 | 이종 도메인 협력 |
## 장단점
| 장점 | 단점 |
|------|------|
| 프라이버시 보호 | 통신 비용 |
| 데이터 이동 불필요 | 비동질적 데이터 문제 |
| 규제 준수 용이 | 악의적 참여자 위험 |
| 분산 컴퓨팅 활용 | 모델 수렴 어려움 |
## 활용 분야
- 의료: 병원 간 협력 학습 (환자 데이터 보호)
- 금융: 사기 탐지 모델 공동 학습
- 모바일: 키보드 예측, 음성 인식
- IoT: 엣지 디바이스 협력 학습
