#AI #에이전트 #프로토콜

## 정의
AI 에이전트 간 통신을 위한 프로토콜, A2A
- 서로 다른 AI 에이전트들이 협력하고 통신할 수 있도록 하는 표준화된 프로토콜
- 멀티 에이전트 시스템에서 에이전트 간 상호작용 지원
## 키워드
* Agent to Agent, 멀티 에이전트, 협력, 통신 프로토콜
## 암기법
* A2A = Agent(에이전트) + to + Agent(에이전트)
* 에이전트 간 수평적 협력
## 연관 토픽
- [[MCP(Model Context Protocol)]] - 모델-도구 연결
- [[LLM]] - 대규모 언어 모델
- [[AI 거버넌스 (AI Governance)]] - 거버넌스

## 핵심 기능
| 기능 | 설명 |
|------|------|
| ==에이전트 발견== | 다른 에이전트 탐색 |
| ==능력 협상== | 수행 가능 작업 교환 |
| ==작업 위임== | 다른 에이전트에 작업 요청 |
| ==결과 공유== | 작업 결과 전달 |
| ==상태 동기화== | 에이전트 간 상태 공유 |
## A2A vs MCP
| 구분 | A2A | MCP |
|------|-----|-----|
| 목적 | 에이전트 간 통신 | 모델-도구 연결 |
| 대상 | Agent ↔ Agent | Model ↔ Tool/Data |
| 관계 | 수평적 협력 | 수직적 연결 |
| 주체 | 자율 에이전트 | LLM 애플리케이션 |
## 멀티 에이전트 시스템
```
┌─────────────────────────────────────────────────┐
│           멀티 에이전트 협력                     │
│                                                  │
│  [사용자 요청]                                  │
│       │                                         │
│       ▼                                         │
│  ┌──────────┐                                  │
│  │Orchestrator│ ← 작업 분배/조율               │
│  └──────────┘                                  │
│       │                                         │
│  ┌────┼────┬────┐                              │
│  ▼    ▼    ▼    ▼                              │
│ [검색][코딩][분석][작성] ← 전문 에이전트        │
│  Agent Agent Agent Agent                        │
└─────────────────────────────────────────────────┘
```
## 활용 사례
- 복잡한 작업의 분산 처리
- 전문 에이전트 협업
- 자율 에이전트 네트워크
- AI 워크플로우 자동화
