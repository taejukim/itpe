## 정의
상속성(Inheritance)
기존 클래스의 속성과 메서드를 상속받아 새로운 클래스를 만드는 객체지향 원칙, 상속성
- 상위 클래스(부모 클래스)의 특성을 하위 클래스(자식 클래스)가 물려받아 재사용하는 기법
- 코드 중복을 제거하고 계층 구조를 통해 클래스 간 관계를 명확히 표현

## 구성요소
- 상위 클래스: 상속을 제공하는 부모 클래스 (Base Class, Super Class)
- 하위 클래스: 상속을 받는 자식 클래스 (Derived Class, Sub Class)
- 메서드 상속: 상위 클래스의 메서드를 하위 클래스에서 사용 가능
- 메서드 오버라이딩: 하위 클래스에서 상위 클래스 메서드를 재정의
- 다중 상속: 여러 상위 클래스로부터 상속받는 것 (일부 언어에서 지원)
- 계층 구조: 클래스 간 상속 관계로 형성되는 계층 구조

## 구성도
상위 클래스 → 상속 → 하위 클래스 → 확장/재정의 → 새로운 기능 추가
