#SW공학 #테스트 #품질관리

## 정의
키워드를 도출하여 테스트에 적용하는 키워드 기반 테스팅
- 기획서와 요구사항에서 키워드를 추출, Testcase에 적용, 수동테스트 및 자동화테스트 수행
특징) 자동/수동 테스트에 적합, ISO 29119
## 키워드
- 키워드 도출, 자동/수동 테스트, 테스트 라이브러리, 저장소, 툴 브릿지
- 단일 키워드 복합 키워드
## 암기법
- 절차 - 정작실리 : 정의, 작성, 실행, 리팩토링
- 유형 - 도중인 : 도메인, 중간, 테스트 인터페이스 계층
