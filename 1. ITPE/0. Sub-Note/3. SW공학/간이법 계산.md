## 정의
소규모 사업 간이 비용 산정, 간이법
- 소규모 정보화 사업(5억원 미만)에 적용하는 간편한 소프트웨어 개발 비용 산정 방법
- 상세 기능점수(FP) 산정 없이 단위 가격과 개발 기간을 기반으로 신속하게 비용 계산

## 구성요소
### 간이법 산정 기준
- 적용 대상: 총사업비 5억원 미만
- 기본 단가: 정부 고시 SW 기술자 단가
- 기술 등급: 특급, 고급, 중급, 초급
- 개발 기간: 월(MM, Man-Month) 단위
- 난이도 계수: 프로젝트 복잡도 반영 (0.8~1.2)
- 간접비: 직접비의 일정 비율 (통상 20~30%)

### 산정 공식
```
총 개발 비용 = 직접비 + 간접비

직접비 = Σ(기술자 단가 × 투입 인월(MM))

간접비 = 직접비 × 간접비율

조정 비용 = 직접비 × 난이도 계수
```

### 기술자 등급별 역할
- 특급: 프로젝트 관리자(PM), 기술 총괄
- 고급: 선임 개발자, 아키텍트
- 중급: 개발자, 설계자
- 초급: 주니어 개발자, 테스터

## 구성도
```
[간이법 산정 프로세스]

1. 사업 범위 정의
   └─ 주요 기능, 화면, 보고서 식별

2. 개발 기간 추정
   └─ 총 개발 소요 기간 (월)

3. 투입 인력 산정
   ├─ 특급: 1 MM
   ├─ 고급: 3 MM
   ├─ 중급: 6 MM
   └─ 초급: 2 MM

4. 단가 적용
   ├─ 특급: 8,000천원/MM
   ├─ 고급: 6,000천원/MM
   ├─ 중급: 4,500천원/MM
   └─ 초급: 3,000천원/MM

5. 직접비 계산
   = (8,000×1) + (6,000×3) + (4,500×6) + (3,000×2)
   = 8,000 + 18,000 + 27,000 + 6,000
   = 59,000천원

6. 간접비 추가
   = 59,000 × 25% = 14,750천원

7. 총 비용
   = 59,000 + 14,750 = 73,750천원

[난이도 계수 적용]

난이도      │ 계수  │ 적용 사례
────────────┼───────┼─────────────
매우 단순   │ 0.8   │ 단순 CRUD
단순        │ 0.9   │ 일반 웹앱
보통        │ 1.0   │ 표준 시스템
복잡        │ 1.1   │ 연동 多
매우 복잡   │ 1.2   │ 고난이도

조정 비용 = 직접비 × 난이도 계수

[간이법 vs 기능점수법]

구분        │ 간이법        │ 기능점수법
────────────┼───────────────┼─────────────
대상        │ 5억 미만      │ 5억 이상
산정 기준   │ 인월(MM)      │ 기능점수(FP)
복잡도      │ 낮음          │ 높음
산정 시간   │ 빠름(1~2일)   │ 느림(1~2주)
정확도      │ 낮음          │ 높음
```

## 키워드
- 5억원 미만, 인월(MM), 기술자 단가
- 특급·고급·중급·초급, 직접비·간접비
- 난이도 계수, 신속 산정
- SW 기술자 노임 단가

## 암기법
- 간이법 대상: 5억 미만
- 기술 등급: 특고중초 (특급-고급-중급-초급)
- 비용 구성: 직간 (직접비-간접비)
