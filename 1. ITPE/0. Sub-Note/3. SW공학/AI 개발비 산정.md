## 정의
인공지능 시스템 개발 비용 산정, AI 개발비 산정
- AI/머신러닝 프로젝트의 특성을 반영한 개발 비용 산정 방법
- 데이터 수집·전처리, 모델 개발·학습·평가, 인프라 비용 등 AI 특화 요소 포함

## 구성요소
### AI 개발 단계별 비용
- 데이터 수집: 데이터 획득, 구매, 크롤링
- 데이터 전처리: 정제, 라벨링, 증강(Augmentation)
- 탐색적 데이터 분석(EDA): 데이터 분석 및 시각화
- 모델 설계: 알고리즘 선택, 아키텍처 설계
- 모델 학습: 훈련, 하이퍼파라미터 튜닝
- 모델 평가: 검증, 테스트, 성능 측정
- 모델 배포: 서빙, API 개발, 인프라 구축
- 모델 모니터링: 성능 추적, 재학습

### 비용 산정 요소
- 데이터 비용: 데이터 수집·구매·라벨링 비용
- 인력 비용: AI 엔지니어, 데이터 사이언티스트, MLOps
- 인프라 비용: GPU/TPU 서버, 클라우드 비용
- 도구 비용: ML 플랫폼, 라벨링 도구, MLOps 도구
- 기술 난이도: 모델 복잡도, 정확도 요구수준
- 데이터 규모: 학습 데이터 양, 차원

### 산정 공식
```
총 AI 개발 비용 = 데이터 비용 + 개발 비용 + 인프라 비용

데이터 비용 = 수집비 + 라벨링비 + 전처리비
라벨링비 = 데이터 건수 × 건당 라벨링 단가

개발 비용 = Σ(인력 × 투입 기간 × 단가)

인프라 비용 = GPU 사용료 + 스토리지 + 네트워크
```

## 구성도
```
[AI 개발 비용 구조]

1. 데이터 비용 (30~40%)
   ├─ 수집: 크롤링, API, 구매
   ├─ 라벨링: 이미지/텍스트 태깅
   │  └─ 이미지: 100~500원/건
   │  └─ 텍스트: 50~300원/건
   └─ 전처리: 정제, 증강, 변환

2. 개발 비용 (40~50%)
   ├─ AI 엔지니어: 8,000천원/MM
   ├─ 데이터 사이언티스트: 9,000천원/MM
   ├─ MLOps 엔지니어: 7,000천원/MM
   └─ 프로젝트 관리자: 8,500천원/MM

3. 인프라 비용 (10~20%)
   ├─ GPU 서버 (A100, V100)
   │  └─ 온프레미스: 5,000~20,000만원
   │  └─ 클라우드: 3~10달러/시간
   ├─ 스토리지 (TB당)
   └─ MLOps 플랫폼

4. 도구·라이선스 (5~10%)
   ├─ 라벨링 도구
   ├─ ML 플랫폼
   └─ 모니터링 도구

[AI 프로젝트 단계별 노력 배분]

데이터 수집·전처리:  30%
모델 개발·학습:      40%
모델 평가·튜닝:      15%
배포·모니터링:       15%

[난이도별 비용 계수]

난이도      │ 계수  │ 적용 사례
────────────┼───────┼──────────────────
단순        │ 1.0   │ 분류, 회귀 (정형)
보통        │ 1.3   │ 이미지 분류, NLP
복잡        │ 1.7   │ 객체 탐지, 생성모델
매우 복잡   │ 2.0   │ LLM, 멀티모달

[산정 예시]

프로젝트: 이미지 분류 모델 개발
데이터: 10,000장 이미지

데이터 비용:
- 수집: 5,000,000원
- 라벨링: 10,000건 × 300원 = 3,000,000원
- 전처리: 2,000,000원
소계: 10,000,000원

개발 비용:
- AI 엔지니어: 6MM × 8,000,000원 = 48,000,000원
- 데이터 과학자: 3MM × 9,000,000원 = 27,000,000원
소계: 75,000,000원

인프라 비용:
- GPU 클라우드: 200시간 × $5 = $1,000 ≈ 1,300,000원
- 스토리지: 1,000,000원
소계: 2,300,000원

총 비용: 87,300,000원
```

## 키워드

- 하이퍼파라미터 튜닝, 모델 배포, 모니터링
