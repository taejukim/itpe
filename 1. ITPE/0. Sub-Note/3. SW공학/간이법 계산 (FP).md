## 구성요소
- FP(Function Point)
	- 데이터 기능
		- ILF : 내부 논리 파일(Internal Logical File)
		- EIF : 외부 연계 파일(External Interface File)
	- 트랜젝션
		- EI : 외부 입력(External Input)
		- EO : 외부 출력(External Output)
		- EQ : 외부 조회(External Query)
- 보정계수 : 
	- 규모 보정계수 : 500FP 미만 시 1.28
	- 연계복잡성 수준 
	- 성능요구수준
	- 운영환경 호환성
	- 보안성 수준
- 단가 : 605,784원

## 특징
- 신속성: 빠른 규모 산정
- 간략성: 복잡한 계산 불필요
- 근사치: 정확도는 상대적으로 낮음
## 절차
- 사전준비 :  간이법 산출물 결정, [[WBS]] 정리
- 개발 FP 산출 : 개발 대산 SW기능 식별 및 기능점수 산정
- 보정 전 원가 산출 : 산정된 기능 점수 단 단가를 곱하여 원가를 산출 
- 보정 후 원가 산출 : SW사업별 특성 고려 보정요소별 보정계수를 곱해 산출
- 직접 경비 이윤 산출 : 개발 원가에 25% 내에서 산출
- SW 개발비 산출 : 개발원가 + 직접경비 + 이윤
## 산정 방식
- DB에 따라 ILF, EIF의 점수 산정 (문제에서 단위점수 주어짐)
- 기능 입력/출력/조회 기능에 따하 EI, EO, EQ 점수 산정
- 보정계수 반영
- 이윤 25% 반영
