#SW공학 #감리 #PMO #필수

## 정의
감리와 PMO 역할 구분, 감리 vs PMO
- 감리는 독립적 검증·확인 활동, PMO는 프로젝트 관리 지원 조직으로 목적과 역할이 상이
- 감리는 사후 점검 중심, PMO는 실시간 관리 지원 중심
## 키워드
- 감리(독립/검증), PMO(소속/지원)
- 사후 점검, 상시 관리
- 적법성/적정성, 효율성/효과성
- 지적 권고, 조정 지원
## 암기법
- 감리: 독검사지 (독립적-검증-사후-지적)
- PMO: 소지상조 (소속-지원-상시-조정)
- 핵심 차이: 독립 vs 소속, 사후 vs 상시
## 구성요소
### 감리(Audit)
- 독립성: 사업자와 독립적 제3자
- 시점: 단계별 또는 사후 점검
- 목적: 적법성, 적정성 검증
- 활동: 문서검토, 현장점검, 지적사항 도출
- 산출물: 감리보고서, 지적사항, 개선방안
- 권한: 지적 및 개선 권고
- 법적 근거: 전자정부법
### PMO(Project Management Office)
- 소속: 발주기관 또는 사업자 소속
- 시점: 사업 전 기간 상시 지원
- 목적: 프로젝트 성공적 수행 지원
- 활동: 계획수립, 진도관리, 이슈관리, 의사결정 지원
- 산출물: 관리계획서, 진척보고서, 이슈리스트
- 권한: 조정 및 지원 (지시 권한 제한적)
- 법적 근거: 계약 또는 내부 규정
## 구성도
```
[감리 vs PMO 비교표]

구분      │ 감리 (Audit)      │ PMO
──────────┼───────────────────┼──────────────────
목적      │ 검증·확인         │ 관리 지원
독립성    │ 독립적 제3자      │ 발주/사업자 소속
시점      │ 단계별/사후       │ 전 기간 상시
관점      │ 적법성/적정성     │ 효율성/효과성
활동      │ 점검·평가         │ 관리·조정
산출물    │ 감리보고서        │ 관리계획서
권한      │ 지적·권고         │ 조정·지원
법적근거  │ 전자정부법        │ 계약
수행자    │ 감리법인·감리원   │ PMO 전문가
사업 참여 │ 단기 (주기적)     │ 장기 (전 기간)
```
