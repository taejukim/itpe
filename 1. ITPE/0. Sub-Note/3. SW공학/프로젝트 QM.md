#SW공학 #품질관리 #프로젝트관리 #테스트 #필수

## 정의
프로젝트 품질 보증 활동, 품질 관리
- 프로젝트 산출물이 품질 요구사항을 만족하도록 계획-보증-통제하는 체계적 관리 활동
- [[PMBoK 8판]]의 지식영역으로, 품질 계획, 품질 보증(QA), 품질 통제(QC)로 구성
## 키워드
- Quality Planning, QA, QC, PDCA
- ISO 9001, CMMI, 품질 표준
- 품질 메트릭, 품질 비용, 지속 개선
- 예방, 평가, 통제
## 암기법
- 품질 관리 3요소: 계보통 (계획-보증-통제)
- QA vs QC: QA(프로세스) vs QC(산출물)
- 품질 비용: 예평내외 (예방-평가-내부실패-외부실패)
## 목적
- 프로젝트 산출물의 품질 요구사항 충족 보장
- 품질 비용 최적화 및 지속적 품질 개선
## 구성요소
- Quality Planning(품질 계획): 품질 표준과 측정 기준 정의
- Quality Assurance(품질 보증, QA): 프로세스가 올바르게 수행되는지 확인
- Quality Control(품질 통제, QC): 산출물이 기준을 만족하는지 검증
- Quality Standard(품질 표준): ISO 9001, CMMI, ISO 25010
- Quality Metrics(품질 메트릭): 결함 밀도, 신뢰도, 가용성
- Quality Audit(품질 감사): 독립적 품질 검토
- PDCA Cycle: Plan-Do-Check-Act 지속 개선
- Cost of Quality(품질 비용): 예방비용, 평가비용, 실패비용
## 구성도
```
[품질 관리 프로세스]

┌────────────────┐
│Quality Planning│ ← 품질 표준, 메트릭 정의
└───────┬────────┘
        │
        ▼
┌────────────────┐
│Quality         │ ← 프로세스 준수 확인
│Assurance (QA)  │    (예방적 활동)
└───────┬────────┘
        │
        ▼
┌────────────────┐
│Quality         │ ← 산출물 검증
│Control (QC)    │    (검출적 활동)
└───────┬────────┘
        │
        └──────> 개선

[PDCA Cycle]

    Plan (계획)
       │
       ▼
    Do (실행)
       │
       ▼
    Check (확인)
       │
       ▼
    Act (조치) ─────┐
       │            │
       └────────────┘ 지속 개선

[품질 비용]

1. 예방 비용 (Prevention Cost)
   - 교육, 프로세스 개선, 도구 구입

2. 평가 비용 (Appraisal Cost)
   - 테스트, 리뷰, 감사

3. 내부 실패 비용 (Internal Failure)
   - 재작업, 결함 수정

4. 외부 실패 비용 (External Failure)
   - 고객 불만, 보증 수리, 소송

예방 비용 ↑ → 실패 비용 ↓
```
