#SW공학 #개발모델 #필수

## 정의
위험 분석을 중심으로 반복적으로 개발하는, 나선형 모델
- 보엠(Boehm)이 제안하였으며, [[폭포수 모델]]과 프로토타입 모델의 장점에 위험 분석을 결합
- 프로젝트 진행 중 발생하는 위험을 최소화하기 위해 여러 번의 나선을 그리며 개발 범위를 확장함
특징) 위험최소화, 점진적구체화, 고수준품질, 개발장기화
## 키워드
* 위험 분석, 반복적 개발, 보엠(Boehm), 대규모 프로젝트, 점진적 상세화
## 암기법
* 계위개고 (계획수립, 위험분석, 개발, 고객평가)
* 나선형은 위험분석이 핵심
## 구성요소
| 단계      | 설명                | 주요 활동           |
| ------- | ----------------- | --------------- |
| 계획 수립   | 목표, 대안, 제약조건 결정   | 요구사항 정의, 목표 설정  |
| 위험 분석   | 대안 평가, 위험 식별 및 해결 | 위험 분석, 프로토타입 개발 |
| 개발 및 검증 | 제품 개발             | 설계, 구현, 테스트     |
| 고객 평가   | 결과 평가, 다음 단계 계획   | 고객 검토, 다음 반복 계획 |
