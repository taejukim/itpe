## 정의
AI 기반 자동 코드 생성, AI 코딩
- 인공지능(AI)과 대규모 언어 모델(LLM)을 활용하여 자연어 명령이나 주석으로 코드를 자동 생성하는 개발 방식
- 개발자의 생산성을 극대화하고 반복 작업을 자동화하여 비즈니스 로직 구현에 집중

## 특징
- 자동화: 코드 자동 생성 및 완성
- 생산성: 개발 속도 대폭 향상
- 직관성: 자연어 기반 인터페이스
- 학습성: AI가 패턴 학습하여 제안

## 목적
- AI 기반 코드 생성으로 개발 생산성 극대화
- 반복 작업 자동화로 창의적 업무에 집중

## 구성요소
- AI Copilot(AI 코파일럿): 실시간 코드 제안 및 자동완성 (GitHub Copilot, Cursor)
- LLM(Large Language Model): GPT-4, Claude, Codex 등 코드 생성 모델
- Prompt Engineering(프롬프트 공학): 효과적인 코드 생성을 위한 자연어 명령 설계
- Code Generation(코드 생성): 함수, 클래스, 테스트 코드 자동 생성
- Code Review(코드 리뷰): AI 기반 코드 품질 검증 및 개선 제안
- Refactoring(리팩토링): AI 기반 코드 구조 개선
- Documentation(문서화): 자동 주석 및 API 문서 생성
