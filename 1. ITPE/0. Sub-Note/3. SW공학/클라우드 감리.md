### 클라우드 전환 단계 별 감리 방법
- 전환준비
	- 전환 계획 수립
	- 구축 계획 수립
- 전환신청
	- nTOPS 계정 신청
	- 방화벽 접근 신청
- 클라우드 전환
	- 상용 SW 설치
	- 보안 취약점 점검
- 서비스 안정화
	- 서비스 전환
	- 서비스 안정화
### 검토 항목
- 전환 적합성 평가
	- 클라우드 전환 가용성
	- 공개 SW 전환 가능 여부
- 인프라및 시스템 분석
	- 장비 사용 만료 여부
	- SW 지원 만료 여부
- Network 및 보안 요구사항
	- 고립망, DR 필수 여부
	- 망간 데이터 연동 여부
