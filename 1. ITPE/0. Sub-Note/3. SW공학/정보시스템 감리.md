## 정의
정보시스템 독립적 검증, 정보시스템 감리
- 정보시스템의 구축·운영이 관련 법령과 기준에 적합하게 수행되는지 독립적으로 검증·확인하는 활동
- 전자정부법 제57조에 근거하여 총사업비 5억원 이상 정보화 사업에 의무 시행

## 구성요소
### 감리 유형
- 사업수행 감리: 사업 전 과정 진행 상황 점검
- 단계별 감리: 주요 단계(분석, 설계, 구축) 점검
- 사전 감리: 사업 착수 전 타당성 검토
- 운영 감리: 운영 단계 적정성 점검
- 유지보수 감리: 유지보수 적정성 점검

### 감리 주체
- 감리법인: 한국지능정보사회진흥원(NIA) 등록 법인
- 감리원: 정보관리기술사, 감리사 자격 보유자
- 발주기관: 감리 요청 기관
- 사업자: 피감리 대상

### 감리 영역
- 사업관리: 계획, 일정, 비용, 위험관리
- 기술: 요구사항, 설계, 개발, 테스트
- 데이터: 데이터 모델링, 마이그레이션
- 보안: 보안 요구사항, 취약점 점검
- 품질: 품질 계획, 테스트, 형상관리

## 구성도
```
[감리 체계]

┌─────────────┐
│  발주기관   │ (감리 요청)
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  감리법인   │ (독립적 검증)
│  - 감리원   │
└──────┬──────┘
       │ 감리 수행
       ▼
┌─────────────┐
│   사업자    │ (피감리 대상)
└─────────────┘

[감리 프로세스]

1. 감리 준비
   └─ 감리 계획 수립, 감리팀 구성

2. 감리 착수
   └─ 킥오프 미팅, 일정 협의

3. 감리 수행
   ├─ 문서 검토
   ├─ 현장 점검
   ├─ 인터뷰
   └─ 산출물 검증

4. 감리 결과
   ├─ 지적사항 도출
   ├─ 개선방안 제시
   └─ 감리보고서 작성

5. 사후관리
   └─ 조치 확인, 이행 점검

[감리 등급]

양호: 지적사항 경미, 즉시 조치 가능
보통: 일부 개선 필요
미흡: 중대한 문제, 재검토 필요

[감리 의무 대상]

총사업비 5억원 이상
AND
(정보화사업 OR 정보시스템 구축)
```

## 키워드
- 전자정부법, 독립적 검증, 감리법인, 감리원
- 사업수행, 단계별, 사전, 운영, 유지보수
- 지적사항, 개선방안, 감리보고서
- 5억원 이상, 의무 시행

## 암기법
- 감리 유형 5개: 사단사운유 (사업수행-단계별-사전-운영-유지보수)
- 감리 영역 5개: 사기데보품 (사업-기술-데이터-보안-품질)
- 프로세스: 준착수결사 (준비-착수-수행-결과-사후)
