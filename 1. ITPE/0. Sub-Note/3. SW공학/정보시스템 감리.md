#SW공학 #감리 #품질보증 #필수

## 정의
독립적 제3자 점검 활동, 정보시스템 감리
- 이해관계 없는 독립자가 정보시스템 구축/운영 점검
## 키워드
- 독립성, 제3자, 의무감리, 5억원
## 암기법
- **목표 4개**: 요안효효 (요건-안전-효율-효과)
- **의무감리**: 5억원 이상
- 준착수보종화 : 준비-착수-수행-보고서 작성 검토-종료-보고서 확정
## 목적
| 목적 | 설명 |
|-----|------|
| 요건준수 | 기준, 법규, 표준 준수 |
| 안전성 | 기밀성, 무결성, 가용성 |
| 효율성 | 자원 충분 활용 |
| 효과성 | 목표 달성 |

## 감리 유형
- ISP, EA, SD, DB, OP, MA 감리
	- ISP (information strategy planning) : 정보화 전략계획 감리
	- EA (Enterprise Architecture) : 전사 아키텍처 감리
	- SD (System Devlopment) : 시스템 개발 관리
	- DB (Database) : 데이터 베이스 감리
	- OP (Operation) : 운영 감리
	- MA (Maintenance) : 유지보수 감리
