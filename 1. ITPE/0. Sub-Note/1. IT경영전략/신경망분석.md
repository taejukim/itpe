## 정의
생물학적 뇌의 구조를 모방한 인공신경망 기반 데이터 분석 기법, 신경망 분석
- 뉴런의 연결 구조를 수학적으로 모델링하여 복잡한 비선형 관계를 학습하고 예측
- 딥러닝의 핵심 기술로, 이미지 인식, 자연어 처리, 수요 예측 등 다양한 분야에 활용

## 목적
- 이미지 인식, 음성 처리, 자연어 이해 등 전통적인 방법으로 해결하기 어려운 비정형 데이터 분석
- 시계열 예측, 금융 시장 분석, 질병 진단 등 정밀한 예측 모델 구축
- 대규모 데이터로부터 특징(Feature)을 자동 추출하여 분석의 효율성 및 정확도 제고
- 강화학습 등과 결합하여 복잡한 환경에서의 최적의 의사결정 모델 구현

## 구성요소
- 입력층(Input Layer): 분석 대상 데이터를 받아들이는 층
- 은닉층(Hidden Layer): 가중치(Weight)와 활성화 함수를 통해 특징을 추출하는 층
- 출력층(Output Layer): 최종 분석 결과를 산출하는 층
- 가중치(Weight): 뉴런 간 연결의 강도를 나타내는 매개변수
- 활성화 함수(Activation Function): 신호의 출력 여부를 결정 (ReLU, Sigmoid 등)
- 역전파(Backpropagation): 오차를 역방향으로 전달하여 가중치를 갱신하는 학습 알고리즘

## 활성화 함수 특징
* 항등함수: 연속적인 값을 예측하는 회귀 문제의 출력층에서 주로 사용
- Sigmoid: 0~1 사이 출력, 기울기 소실(Vanishing Gradient) 문제 존재
- ReLU: 0보다 작으면 0, 크면 x 출력, 연산 속도가 빠르고 기울기 소실 문제 완화
- Softmax: 다중 분류 시 출력층에서 확률값으로 변환하여 사용

## 구성도 요소
* 아키텍처: 다층 퍼셉트론(MLP) 구조 중심
* 관계: 학습 데이터를 통한 순전파(Forward)와 오차 수정을 위한 역전파(Backward) 반복
```
[Input] -> [Hidden Layer 1] -> [Hidden Layer 2] -> [Output]
   |             |                  |              |
   +-------------<---- (Backpropagation) ----------+
```

## 키워드
* 인공신경망(ANN), 딥러닝, 퍼셉트론, 활성화 함수, 역전파

## 암기법
* 입은출가활 (입력층, 은닉층, 출력층, 가중치, 활성화함수)
* 항시렐 (항등함수, 시그모이드, 렐루)
