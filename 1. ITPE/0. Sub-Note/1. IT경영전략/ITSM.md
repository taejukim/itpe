## 정의
사용자에게 고품질 IT 서비스를 제공하기 위한 통합 관리 체계, ITSM
- IT 중심이 아닌 고객 및 비즈니스 관점에서 효율적인 서비스를 운영하고 관리하는 기법
- 표준화된 프로세스, 숙련된 인력, 최적화된 기술을 결합하여 서비스 수준([[SLA]])을 보장함

## 구성요소
- ==인력(People): IT 서비스를 운영하고 관리하는 전문 인력==
- ==조직(Organization): IT 서비스 제공을 위한 전담 조직 및 역할/책임(R&R)==
- ==기술(Technology): ITSM 구축을 위한 관리 도구 및 기술 (AI, 클라우드 등)==
- ==프로세스(Process): ITIL 등 표준 프레임워크에 기반한 표준화된 업무 절차==

## 구성도
* 운영 구조: 서비스 요청(SR) → 서비스 데스크 → 프로세스 처리(장애/변경 등) → 피드백
* 관계: ITIL을 프레임워크로 채택하고 ISO 20000 인증을 통해 수준 검증
```
[ITSM 지원 기술]
            (Cloud, 블록체인, 빅데이터, 가상화)
                          │ 기술적용
                          ▼
┌──────────────┐   ┌──────────────┐   ┌───────────────────┐
│ Best Practice│   │     ITSM     │   │  프로젝트 관리/SW │
│ (ITIL, eSCM, │ ─▶│ (기업의 IT   │◀─ │  품질인증 (CMMI,  │
│  ISO 20000)  │   │ 서비스 관리)  │   │  SPICE, PMO, QA)  │
└──────────────┘   └──────────────┘   └───────────────────┘
                          ▲
                          │
                  [서비스 수준/가용성 측정]
                      (SoW, SLM, SLA)
```
