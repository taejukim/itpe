#IT경영전략 #정보전략계획 #필수

## 정의
ISP (Information Strategy Planning) 9.0
조직의 전략 기획 핵심, ISP의 개요
- 조직의 디지털, 데이터, 보안 전략 통합 설계를 위한 종합 정보화 전략
특징) 비즈니스 연계성, 전사적 통합성, 미래지향적
## 키워드
* 전략적 연계(Alignment), To-Be 모델, 로드맵, 우선순위, 정보화 투자
## 암기법
* 환현비목통 (환경분석, 현황분석, 비전 수립, 목표모델, 통합이행계획)
## 목적
- 경영 전략과 IT 전략의 일관성 확보 및 비즈니스 가치 창출
- 전사적 관점의 정보시스템 통합 및 자원 배분 최적화
- 정보화 투자의 타당성 검토 및 실행 가능한 이행 로드맵 수립
- 정보기술 인프라 및 [[IT Governance]] 체계의 고도화 방향 제시
- 우리 회사는 어떤 AI 서비스를 도입할 것인가?
## 구성요소
- ==환==경 분석: SWOT, PEST, 5Forces, 7S, 벤치마킹 등을 통한 내외부 환경 파악
- ==현==황 분석: 현행 업무 프로세스 및 정보시스템(As-Is) 분석, 이슈 도출
- 정보화 ==비==전 및 전략: 목표 모델(To-Be) 설정, 정보화 비전 및 추진 전략 수립
- ==목==표 아키텍처 설계: 비즈니스, 데이터, 애플리케이션, 기술 아키텍처(EA) 설계
- ==통==합 이행 계획: 이행 과제 상세화, 우선순위 결정, 로드맵, 총사업비 산출
## 구성도 요소
* 추진 절차: 착수 → 분석 → 설계 → 이행 계획 수립
* 연계 구조: 비즈니스 전략(BSP) → 정보화 전략(ISP) → 실행(SI/SM)
```
[단계 1: 환경 분석]      [단계 2: 현행 분석]      [단계 3: 비전 수립]    [단계 4: 이행 계획]
 (Environment)         (As-Is Analysis)       (To-Be Model)        (Implementation)
┌────────────────┐     ┌────────────────┐     ┌────────────────┐     ┌────────────────┐
│  경영 전략 분석   │     │  업무 프로세스     │     │  목표 아키텍처  │        │  통합 이행 과제  │
│  대외 환경 분석   │ ──> │  정보 시스템       │ ──> │  데이터/HW/SW   │ ──> │  우선순위 결정   │
│  IT 동향 분석    │     │  인프라 분석       │     │  미래 운영 체계  │      │  투자 대비 효과  │
└───────┬────────┘     └───────┬────────┘     └───────┬────────┘     └───────┬────────┘
        │                      │                      │                      │
        └──────────────────────┴──────────┬───────────┴──────────────────────┘
                                          ▼
                             [ 정보전략 로드맵 (ISP Report) ]
```
