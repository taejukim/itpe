#IT경영전략 #디지털윤리 #필수

## 정의
소비자를 기만하는 UI/UX 설계, 다크패턴
- 서비스 제공자의 이익을 위해 사용자의 의사결정을 왜곡하거나 제약하는 비윤리적인 디자인
- 전자상거래, 앱 결제 등에서 사용자의 부주의를 악용하여 구매나 구독을 유도함
## 키워드
* 기만적 디자인, 인지 편향, 소비자 보호, 디지털 윤리, UI/UX 규제
## 암기법
- 숨압오방
	- 숨은갱신, 압박형, 오도형, 방해형
## 목적
- 가입은 쉽게, 해지는 어렵게 유도하여 유료 회원 수 및 구독 유지율 증대
- 숨겨진 비용 청구나 옵션 강제 선택을 통한 매출액 극대화
- 개인정보 수집 시 모호한 표현이나 UI 배치를 활용해 과도한 정보 획득
- 다른 선택지를 의도적으로 숨기거나 시각적으로 차별하여 특정 상품 선택 유도
## 구성요소
- 숨은 갱신 : 소비자가 알아 채기 어려운 조작으로 예상치 못한 지출
- 압박형 : 변경을 반복적으로 압박
- 오도형 : 거짓이나 소비자의 착각 실수
- 방해형 : 과도하게 노력을 들여 하게 하는 실수
## 구성도 요소
* 메커니즘: 사용자 인지 편향(Cognitive Bias) 이용 → 부주의 유도 → 비자발적 동의 → 이익 창출
* 관계: 디지털 리터러시 강화 및 소비자 보호법(다크패턴 방지법) 등 법적 규제 확대 중
```

      |                                     |

```
