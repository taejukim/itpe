#IT경영전략 #커머스혁신

## 정의
공간에서 자연스러운 소비행위, 앰미언트 커머스
- IoT, AI, 센서 기술을 활용하여 사용자가 의식하지 못하는 사이에 상거래가 일어나는 환경
- 물리적 매장이나 주거 공간 자체가 쇼핑 채널이 되는 '생활 밀착형' 커머스
- 특징) 상황 인지, 무마찰 결제, 비가시성 인터페이스, 초개인화
## 키워드
* Context-Awareness, 비가시적 경제, 무마찰 결제, 아마존 고, 환경 지능
## 암기법
* 지결개유보 (환경지능, 결제자동화, 개인화, UI유연성, 자동보충)
## 목적
- 구매 프로세스상의 허들(탐색, 결제 등)을 최소화하여 고객 경험 극대화
- 실시간 상황 데이터를 기반으로 최적의 시점에 개인화된 제안 및 자동 주문 수행
- 온라인과 오프라인의 경계를 허무는 끊김 없는(Seamless) 옴니채널 환경 구축
- 고객의 라이프스타일 깊숙이 침투하여 브랜드 로열티 및 재구매율 향상
## 구성요소
- 환경 지능(Ambient Intelligence): 센서를 통해 사용자의 맥락과 주변 상황을 실시간 파악
- 무마찰 결제(Frictionless Payment): 결제 과정을 생략하거나 자동화 (예: 아마존 고)
- 개인화 추천: 위치 및 행동 패턴 데이터를 분석하여 실시간으로 최적의 상품 제안
- 비가시적 UI: 음성 명령, 동작 인식 등 별도의 단말 조작이 필요 없는 인터페이스
- 자동 보충(Auto-replenishment): 생필품 소진 시 자동으로 주문 및 배송 수행
## 구성도 요소
* 아키텍처: 물리적 공간(IoT 센서) → 클라우드 분석(AI) → 자동 서비스(커머스)
* 관계: O2O(Online to Offline)를 넘어선 온-오프라인의 완전한 융합(Phygital)
```
[User Context] -- (Sensor Data) --> [Contextual Intelligence]
       ^                                     |
       |                               (Auto Transaction)
       +------- [Seamless Experience] <------+
```

## 기술 요소
- 센싱 기술: 비콘(Beacon), RFID, 카메라 기반 컴퓨터 비전, 압력 센서
- 인공지능: 고객 행동 패턴 학습, 수요 예측 알고리즘, 음성 인식(Voice AI)
- 인프라: IoT 기기(스마트 냉장고 등), 엣지 컴퓨팅, 고속 무선 네트워크(5G)
- 무마찰 결제: 자동 결제 시스템(Just Walk Out), 생체 인식 결제
- 적용 사례: 아마존 고(Amazon Go), 대시 버튼(Dash Button), 스마트 가전 자동 재주문
