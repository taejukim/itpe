## 정의
위험 대응 계획 수립
식별된 위험에 대한 대응 전략과 계획을 수립하는 활동, 위험 대응 계획
- 위험 분석 결과를 바탕으로 각 위험에 대한 구체적인 대응 방안을 수립
- 위험 발생 시 신속하고 효과적으로 대응할 수 있도록 사전에 계획을 마련

## 목적
- 분석된 위험 요인별로 최적의 대응 전략(회피, 전가, 완화, 수용 등)을 사전에 마련하여 위기 상황 발생 시 신속하게 대처
- 위험으로 인한 부정적 영향은 최소화하고, 기회로 인한 긍정적 영향은 극대화하여 프로젝트 목표 달성 보장
- 대응 계획에 따른 예비비와 자원을 사전에 확보하여 프로젝트의 안정성과 회복탄력성(Resilience) 강화

## 구성요소
| 구분     | 전략           | 설명                |     |
| ------ | ------------ | ----------------- | --- |
| 부정적 위험 | 회피(Avoid)    | 위험을 완전히 제거하거나 회피  |     |
| (위협)   | 전가(Transfer) | 제3자에게 이전 (보험, 계약) |     |
|        | 완화(Mitigate) | 확률이나 영향을 줄임       |     |
|        | 수용(Accept)   | 위험을 수용하고 대비       |     |
| 긍정적 위험 | 활용(Exploit)  | 기회를 확실히 실현        |     |
| (기회)   | 공유(Share)    | 파트너와 기회 공유        |     |
|        | 강화(Enhance)  | 확률이나 영향을 높임       |     |
|        | 수용(Accept)   | 기회를 수용            |     |
