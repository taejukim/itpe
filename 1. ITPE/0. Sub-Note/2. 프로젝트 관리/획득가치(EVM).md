## 정의
일정, 원가, 범위를 통합하여 프로젝트 성과를 측정하는 기법, EVM
- 계획 대비 실제 수행된 가치를 화폐 단위로 환산하여 프로젝트의 현재 상태를 진단하고 미래를 예측
- 객관적인 수치를 통해 일정 지연 및 원가 초과 여부를 조기에 파악함

## 목적
- 일정과 비용 성과를 동일한 척도(통화 단위)로 통합하여 측정함으로써 프로젝트의 실제 상태를 객관적이고 정량적으로 파악
- 계획(PV) 대비 실제 수행 성과(EV)와 비용(AC)을 비교 분석하여 일정 및 비용의 효율성(SPI, CPI) 평가
- 현재 추세를 바탕으로 완료 시점의 예상 비용(EAC)과 기간을 예측하여 선제적인 성과 관리 및 의사결정 지원


## 구성요소
- 계획요소
	- BAC : 프로젝트 전체 예산
- 측정요소
	- PV(계획가치), EV(획득가치), AC(실제원가)
- 분석요소
	- 편차 분석: 
		- CV(원가편차 = EV - AC)
		- SV(일정편차 = EV - PV)
	- 효율 지수: 
		- CPI(원가성과지수 = EV / AC)
		- SPI(일정성과지수 = EV / PV)
- 예측요소
	- 미래 예측: 
		- ETC(완료까지추정치) : BAC-EV/CPI
		- EAC(완료시점추정치) : ETC + AC
		- VAC(완료시점편차) : BAC - EAC
	- TCPI(To-Complete Performance Index): 목표 달성을 위해 필요한 성과 지수
		- (BAC-EV)/(BAC-AC)

## 구성도 요소
* S-Curve: 시간 경과에 따른 PV, EV, AC의 누적 흐름을 시각화
* 관계: 지수가 1보다 작으면 부정적(지연/초과), 1보다 크면 긍정적(조기/절감)

```
  [성과 지표 해석]
  ┌────────────────────────────────────────────┐
  │ CV > 0, CPI > 1 : 예산 절감 (비용 양호)     │
  │ CV < 0, CPI < 1 : 예산 초과 (비용 불량)     │
  │ SV > 0, SPI > 1 : 일정 앞섬 (일정 양호)     │
  │ SV < 0, SPI < 1 : 일정 지연 (일정 불량)     │
  └────────────────────────────────────────────┘
  
  [예측 지표]
  ┌────────────────────────────────────────────┐
  │ BAC (총 예산) = 100                         │
  │ CPI = 0.8                                   │
  │ EAC = BAC / CPI = 100 / 0.8 = 125          │
  │ → 완료 시 125가 필요 (25 초과 예상)         │
  └────────────────────────────────────────────┘
```
