## 정의
획득가치(EVM)
프로젝트의 일정과 비용 성과를 통합적으로 측정하는 관리 기법, Earned Value Management
- 계획된 작업량 대비 실제 완료된 작업량과 실제 비용을 비교하여 성과를 측정
- 프로젝트의 현재 상태를 객관적으로 파악하고 미래 성과를 예측하는 핵심 도구

## 목적
- 일정과 비용 성과를 동일한 척도(통화 단위)로 통합하여 측정함으로써 프로젝트의 실제 상태를 객관적이고 정량적으로 파악
- 계획(PV) 대비 실제 수행 성과(EV)와 비용(AC)을 비교 분석하여 일정 및 비용의 효율성(SPI, CPI) 평가
- 현재 추세를 바탕으로 완료 시점의 예상 비용(EAC)과 기간을 예측하여 선제적인 성과 관리 및 의사결정 지원

## 구성요소
| 지표 | 영문명 | 계산식 | 설명 |
|------|--------|--------|------|
| PV | Planned Value | - | 계획된 작업의 예산 비용 |
| EV | Earned Value | - | 실제 완료된 작업의 예산 비용 |
| AC | Actual Cost | - | 실제 발생한 비용 |
| CV | Cost Variance | EV - AC | 비용 차이 (+: 절감, -: 초과) |
| SV | Schedule Variance | EV - PV | 일정 차이 (+: 앞섬, -: 지연) |
| CPI | Cost Performance Index | EV / AC | 비용 성과 지수 (>1: 양호) |
| SPI | Schedule Performance Index | EV / PV | 일정 성과 지수 (>1: 양호) |
| EAC | Estimate at Completion | BAC / CPI | 완료 시 예상 총 비용 |
| ETC | Estimate to Complete | EAC - AC | 완료까지 예상 비용 |

## 구성도
```
  [EVM 기본 개념]
  
  비용
   ▲
   │                              ┌─── AC (실제 비용)
   │                         ┌───/
   │                    ┌───/
   │               ┌───/        ┌─── PV (계획 가치)
   │          ┌───/         ┌──/
   │     ┌───/          ┌──/
   │  ──/           ┌──/    ┌─── EV (획득 가치)
   │            ┌──/    ┌──/
   │        ┌──/    ┌──/
   │    ┌──/    ┌──/
   │───/────┬──/
   │        │                                    
   └────────┴────────────────────────────▶ 시간
            현재
  
  [성과 지표 해석]
  ┌────────────────────────────────────────────┐
  │ CV > 0, CPI > 1 : 예산 절감 (비용 양호)     │
  │ CV < 0, CPI < 1 : 예산 초과 (비용 불량)     │
  │ SV > 0, SPI > 1 : 일정 앞섬 (일정 양호)     │
  │ SV < 0, SPI < 1 : 일정 지연 (일정 불량)     │
  └────────────────────────────────────────────┘
  
  [예측 지표]
  ┌────────────────────────────────────────────┐
  │ BAC (총 예산) = 100                         │
  │ CPI = 0.8                                   │
  │ EAC = BAC / CPI = 100 / 0.8 = 125          │
  │ → 완료 시 125가 필요 (25 초과 예상)         │
  └────────────────────────────────────────────┘
```

## 암기법
- 계획실제비일: PV(계획), EV(실제 완료), AC(실제 비용), CV(비용 차이), SV(일정 차이)
- CPI, SPI: 1보다 크면 양호, 1보다 작으면 불량
