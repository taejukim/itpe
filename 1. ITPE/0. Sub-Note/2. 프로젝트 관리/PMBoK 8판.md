## 정의
AI와 [[디지털 전환(DX)]] 시대의 지능형 프로젝트 관리 표준, PMBOK 8판
- 7판의 원칙 중심 기조를 유지하면서 ==생성형 AI 활용==, 데이터 기반 의사결정 등 최신 트렌드 강화
- 불확실성이 극대화된 환경에서 프로젝트의 민첩성과 회복탄력성(Resilience)을 강조함

## 6가지 원칙
- 가치중심 : 프로젝트가 아닌 가치 중심
- 이해관계자 중심 : 이해관계자 만족 유지
- 팀역량/리더쉽 : PM이 촉진자 역할
- 적응성 : 유연하게 대응
- 품질 내재화 : 품질 고려 프로젝트 관리
- 책임윤리 : EGS 고려
   
## 7가지 성과 도메인
- 거버넌스 : 프로젝트 착수 ~ 종료 관리 → 사업관리계획서
- 범위 : 프로젝트 범위 결정 → 업무범위서
- 일정 : 프로젝트 단위작업 일정 수립 → 개발계획표
- 재정 : 작업단위별 비용관리 → 원가 기준서
- 이해관계자 : 이해관계자 식별, 참여, 관리 → 관리계획서
- 자원 : 인적, 물적자원 관리 → 물품관리대장
- 위험 : 리스크 대책 수립 → 리스크 관리 계획서

## 구성요소
- AI 통합 관리: PM 작업의 효율화를 위한 AI 도구 활용 및 거버넌스 수립
- 데이터 기반 분석: 실시간 성과 측정 및 예측 분석을 통한 선제적 대응
- 회복탄력성(Resilience): 변화와 장애에 유연하게 대응하고 빠르게 회복하는 능력
- 하이퍼 애자일: 급변하는 시장에 대응하기 위한 초민첩 프로젝트 운영 체계
- 가치 극대화: 단순 산출물(Output) 인도를 넘어 비즈니스 결과(Outcome) 창출 집중

## 구성도 요소
* 진화 방향: Value-based(7판) → Intelligence & Resilience-based(8판)
* 관계: 디지털 에코시스템 내에서의 프로젝트 관리 역량 강조
```
[Traditional PM] -> [Agile PM] -> [AI-driven/Intelligent PM]
                                          |
                                [Data / AI / Resilience]
```
