## 정의
MSA 서비스 서비스 메쉬(Service Mesh)
마이크로서비스 간 통신을 관리하는 인프라 레이어, Service Mesh
- 마이크로서비스 간의 통신, 보안, 관찰성을 처리하는 전용 인프라 계층
- 애플리케이션 코드와 분리하여 서비스 간 통신을 투명하게 관리

## 구성요소
- 사이드카 프록시: 각 서비스와 함께 배포되는 프록시 (Envoy, Linkerd 등)
- 제어 평면: 서비스 메쉬를 관리하고 설정하는 중앙 제어 시스템
- 데이터 평면: 실제 서비스 간 트래픽을 처리하는 프록시 네트워크
- 트래픽 관리: 로드 밸런싱, 라우팅, 트래픽 분할
- 보안: 서비스 간 mTLS, 인증, 인가
- 관찰성: 메트릭, 로그, 트레이싱을 통한 모니터링

## 구성도
애플리케이션 서비스 → 사이드카 프록시 → 서비스 메쉬 네트워크 → 다른 서비스
