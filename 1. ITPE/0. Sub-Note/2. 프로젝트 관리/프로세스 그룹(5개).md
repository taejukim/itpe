## 정의
프로세스 그룹(5개)
프로젝트 생명주기를 구성하는 5개의 주요 프로세스 그룹, 프로세스 그룹
- 프로젝트의 시작부터 종료까지 단계별로 수행되는 프로세스들을 그룹화한 것
- 각 프로세스 그룹은 순차적이거나 반복적으로 수행되며, 프로젝트 관리의 기본 프레임워크
- [[ISO 21500]], [[PMBoK 7판]], [[PMBoK 8판]]에서 표준화된 프로세스 그룹

## 구성요소
- 착수 프로세스 그룹(Initiating Process Group): 프로젝트 승인 및 착수, 초기 이해관계자 식별
- 기획 프로세스 그룹(Planning Process Group): 프로젝트 목표 정의, 범위·일정·비용·품질 계획 수립
- 실행 프로세스 그룹(Executing Process Group): 프로젝트 계획 실행, 팀 관리, 의사소통, 품질 보증
- 모니터링 및 통제 프로세스 그룹(Monitoring & Controlling Process Group): 진행 상황 추적, 변경 통제, 위험 모니터링
- 종료 프로세스 그룹(Closing Process Group): 프로젝트 완료, 산출물 인수, 문서화, 자원 해제

## 구성도
착수 → 기획 → 실행 → 모니터링 및 통제 → 종료
(각 단계는 순차적이거나 반복적으로 수행)

## 키워드
- 착수, 기획, 실행, 모니터링, 통제, 종료, 프로세스 그룹
