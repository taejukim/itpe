## 정의
번다운 차트
Sprint 동안 남은 작업량을 시간에 따라 시각화한 차트, Burn-down Chart
- Sprint 진행 상황을 한눈에 파악할 수 있도록 남은 작업량을 그래프로 표현
- 계획된 완료 시점과 실제 진행 속도를 비교하여 Sprint 목표 달성 여부를 예측
- [[Agile 프로세스 Scrum]]에서 Sprint 진행 상황을 모니터링하는 주요 도구

## 구성요소
- X축: Sprint 일자
- Y축: 남은 작업량 (Story Point 또는 시간)
- 이상선(Ideal Line): 계획대로 진행될 경우의 예상 완료 선
- 실제선(Actual Line): 실제 진행 상황을 나타내는 선
- 작업량 추정: 각 작업의 작업량을 Story Point나 시간으로 추정
- 일일 업데이트: 매일 남은 작업량을 업데이트하여 추세 파악

## 구성도
Sprint 시작 → 작업량 추정 → 이상선 그리기 → 일일 업데이트 → 실제선 그리기 → 완료 시점 예측
