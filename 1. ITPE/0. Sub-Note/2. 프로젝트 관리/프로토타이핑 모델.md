## 정의
프로토타이핑 모델
고객 요구사항이 불명확할 때 초기 모형을 제작하여 고객과 의사소통을 하여 조정하는 개발 모델, 프로토타이핑 모델
- 빠르게 프로토타입을 만들어 고객 피드백을 받아 요구사항을 명확히 하는 반복적 접근법
- 요구사항이 불명확하거나 시스템 이해도가 낮은 경우에 효과적인 개발 방법론

## 구성요소
- 프로토타입 정의: 프로토타입의 목적과 범위 정의
- 요구사항 수집: 고객으로부터 초기 요구사항 수집
- 프로토타입 설계: 프로토타입의 기본 설계
- 프로토타입 개발: 빠르게 프로토타입 구현
- 고객 평가: 고객이 프로토타입을 평가하고 피드백 제공
- 피드백: 피드백을 반영하여 프로토타입 개선 또는 최종 시스템 개발

## 구성도
프로토타입 정의 → 요구사항 수집 → 설계 → 개발 → 고객 평가 → 피드백 (반복)

## 키워드
