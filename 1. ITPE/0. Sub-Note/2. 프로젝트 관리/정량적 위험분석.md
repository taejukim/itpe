## 정의
정량적 위험분석
위험의 영향을 수치적으로 측정하고 분석하는 활동, 정량적 위험분석
- 위험의 확률과 영향도를 수치화하여 프로젝트 목표에 미치는 영향을 정량적으로 평가
- 의사결정에 필요한 객관적 데이터를 제공하여 위험 대응 우선순위를 결정

## 구성요소
- 확률 분포: 위험 발생 확률의 분포 분석
- 시뮬레이션(몬테카를로): 확률 분포를 이용한 프로젝트 결과 시뮬레이션
- 민감도 분석: 위험 요인이 프로젝트 목표에 미치는 영향도 분석
- 의사결정 나무: 의사결정 시나리오별 기대값 계산
- 기대 가치 분석: 위험의 확률과 영향도를 곱한 기대값 계산
- 위험 등급: 정량적 분석 결과를 바탕으로 위험 등급 분류

## 구성도
정성적 분석 결과 → 확률 분포 설정 → 시뮬레이션 실행 → 민감도 분석 → 의사결정
