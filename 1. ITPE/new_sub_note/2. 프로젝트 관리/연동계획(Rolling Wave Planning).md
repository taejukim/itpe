## 정의
정보가 명확해짐에 따라 계획을 점진적으로 구체화하는 기법, 연동계획
- 가까운 미래의 작업은 상세하게 계획하고, 먼 미래의 작업은 상위 수준에서 계획하는 방식
- 불확실성이 높은 프로젝트에서 점진적 상세화(Progressive Elaboration)를 실현하는 도구

## 구성요소
- 상세 계획 영역: 즉시 실행 가능한 수준으로 작업 패키지를 분할하고 일정 수립
- 개괄 계획 영역: 먼 미래의 작업으로, 마일스톤이나 상위 수준의 요약 활동으로 관리
- 점진적 상세화: 프로젝트 진행에 따라 정보가 가시화되면 개괄 계획을 상세 계획으로 업데이트
- 불확실성 관리: 미래의 예측 불가능한 위험에 유연하게 대응
- 반복적 주기: 일정 주기마다 계획을 검토하고 구체화하는 활동 수행

## 구성도 요소
* 메커니즘: 시간의 흐름에 따라 상세 계획의 범위가 뒤로 밀려나며 확장되는 형태
* 관계: 애자일 방법론의 이터레이션 계획이나 릴리스 계획과 유사한 철학 공유
```
[상세 계획] | [개괄 계획] | [미정]
     |      |            |
     +-----> 시간 경과 ---->
            | [상세 계획] | [개괄 계획]
```

## 키워드
* 점진적 상세화, 불확실성, 파동형 계획, 반복적 업데이트

## 암기법
* 가상멀개 (가까운곳 상세히, 멀리있는곳 개괄적)
