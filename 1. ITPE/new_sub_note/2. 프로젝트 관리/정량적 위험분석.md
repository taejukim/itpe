## 정의
식별된 위험이 프로젝트 목표에 미치는 영향을 수치화하는 기법, 정량적 위험분석
- 위험의 우선순위에 따라 프로젝트의 일정, 비용 등에 미치는 금전적·수치적 영향을 모델링
- 의사결정에 필요한 객관적 근거를 제공하며, 주로 복잡하거나 대규모 프로젝트에서 수행함

## 구성요소
- EMV(Expected Monetary Value): 기대통합가치 (발생 확률 x 영향 금액)
- 민감도 분석(Sensitivity Analysis): 어떤 위험이 가장 큰 영향을 주는지 파악 (Tornado Diagram)
- 의사결정 트리(Decision Tree): 여러 대안의 기대 가치를 비교하여 최적안 선택
- 시뮬레이션(Monte Carlo): 확률 분포를 이용하여 프로젝트 결과의 가능성 예측
- 영향 다이어그램(Influence Diagram): 변수 간의 인과 관계와 영향력을 시각화

## 구성도 요소
* 아키텍처: 정성적 분석(High Risk) → 정량적 모델링 → 통계 분석 → 결과 도출
* 산출물: 프로젝트 목표 달성 확률, 필요한 예비비(Reserve) 규모 산출
```
[Risk Data] -> [Monte Carlo Simulation] -> [Probability Distribution]
                                                 |
                                         [Risk Contingency]
```
