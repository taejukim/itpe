## 정의
프로젝트 관리에 관한 국제 표준 가이드라인, ISO 21500
- PMBOK의 체계를 바탕으로 프로젝트 관리의 기본 원칙과 프로세스를 전 세계적으로 표준화한 문서
- 프로젝트 성공률을 높이기 위해 [[프로세스 그룹(5개)]](5개)과 주제 그룹(10개) 간의 관계를 정의함

## 구성요소
- [[프로세스 그룹(5개)]](5개): 착수, 기획, 실행, 통제, 종료
- 주제 그룹(10개): 통합, 이해관계자, 범위, 자원, 시간, 비용, 위험, 품질, 조달, 의사소통
- 핵심 주체: 프로젝트 스폰서, 프로젝트 매니저(PM), 프로젝트 팀, 이해관계자
- 성과 지표: 프로젝트 목적 달성도, 품질 준수, 일정 및 예산 관리

## 구성도 요소
* 아키텍처: 가로축(프로세스 그룹) x 세로축(주제 그룹) 매트릭스 구조
* 관계: 상위 수준의 지침(Guidance)을 제공하며 업종에 관계없이 범용적으로 적용 가능
```
      [프로세스 그룹]
착수 -> 기획 -> 실행 -> 통제 -> 종료
      |    | 주제 그룹 (10개) |
      +-----------------------+
```

## 키워드
* 국제 표준, PMBOK 기반, 프로세스 매트릭스, 거버넌스, 성과 보증

## 암기법
* 착기실통종 (착수, 기획, 실행, 통제, 종료)
* 통이범자 시비위품조의 (통합, 이해관계자, 범위, 자원, 시간, 비용, 위험, 품질, 조달, 의사소통)
