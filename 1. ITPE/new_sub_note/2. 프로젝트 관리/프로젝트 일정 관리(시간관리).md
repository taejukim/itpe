## 정의
프로젝트를 적기에 완료하기 위해 필요한 프로세스를 관리하는 활동, 일정관리
- 프로젝트 완수를 위해 수행해야 할 활동들의 순서를 정하고 기간을 추정하여 일정을 수립 및 통제
- 제한된 자원 내에서 효율적인 경로를 찾아 프로젝트 완료 목표일을 준수함

## 구성요소
- 활동 정의: [[WBS]]의 작업 패키지를 실행 가능한 수준의 '활동(Activity)'으로 분해
- 활동 순서 배열: 활동 간의 의존 관계(FS, SS, FF, SF) 및 리드/래그 적용
- 활동 기간 추정: 전문가 판단, 유사 추정, 모수 추정, 3점 추정 등을 활용한 기간 산정
- 일정 개발: 네트워크 분석([[CPM(임계경로 관리)]]), 자원 최적화, 일정 단축 등을 통해 일정 모델 수립
- 일정 통제: 일정 기준선 대비 차이를 모니터링하고 일정을 업데이트

## 구성도 요소
* 프로세스: 계획 수립 → 활동 정의 → 순서 배열 → 기간 추정 → 일정 개발 → 일정 통제
* 도구: 간트 차트(Gantt Chart), 네트워크 다이어그램(PDM), 마일스톤 차트
```
[Activity] -> [Sequence] -> [Estimation] -> [Schedule] -> [Control]
```

## 키워드

## 암기법
