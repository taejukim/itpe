## 정의
일정, 원가, 범위를 통합하여 프로젝트 성과를 측정하는 기법, EVM
- 계획 대비 실제 수행된 가치를 화폐 단위로 환산하여 프로젝트의 현재 상태를 진단하고 미래를 예측
- 객관적인 수치를 통해 일정 지연 및 원가 초과 여부를 조기에 파악함

## 구성요소
- 핵심 지표: PV(계획가치), EV(획득가치), AC(실제원가)
- 편차 분석: CV(원가편차 = EV - AC), SV(일정편차 = EV - PV)
- 효율 지수: CPI(원가성과지수 = EV / AC), SPI(일정성과지수 = EV / PV)
- 미래 예측: EAC(완료시점추정치), ETC(완료까지추정치), VAC(완료시점편차)
- TCPI(To-Complete Performance Index): 목표 달성을 위해 필요한 성과 지수

## 구성도 요소
* S-Curve: 시간 경과에 따른 PV, EV, AC의 누적 흐름을 시각화
* 관계: 지수가 1보다 작으면 부정적(지연/초과), 1보다 크면 긍정적(조기/절감)
```
Value |         / AC (Actual Cost)
      |        / EV (Earned Value)
      |       / PV (Planned Value)
      +------------------ Time
```

## 키워드
* S-Curve, CPI/SPI, 편차 분석, 미래 예측(EAC), 정량적 성과 측정

## 암기법
* 피이에이 (PV, EV, AC)
* 이마아 (CV = EV - AC), 이마피 (SV = EV - PV)
