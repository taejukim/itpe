## 정의
실시간 통합 생산 현장 관리 시스템, MES
- 공장의 계획부터 생산 완료까지 현장의 실시간 데이터를 수집하고 통제하는 관리 시스템
- 상위의 ERP(전사적 자원관리)와 하위의 설비 제어기(PLC/HMI) 간의 가교 역할 수행

## 구성요소
- 생산 계획: 제품별 투입 지시 및 가동 시간 관리
- 품질 관리: 불량 원인 분석, 통계적 공정 관리(SPC)
- 자원 관리: 원자재 투입 추적, 설비 유지 보수 및 가동 상태 모니터링
- 작업 추적: 생산 이력 관리(Traceability), 공정 데이터 수집
- 성과 분석: 사이클 타임 분석, 설비 종합 효율(OEE) 측정

## 구성도 요소
* 아키텍처: 계층 구조 (Layer 1: 설비 → Layer 2: MES → Layer 3: ERP)
* 관계: 스마트 팩토리의 핵심 인프라로 수직적 통합 기능 제공
```
[Enterprise Layer (ERP)]
          |
[Execution Layer (MES)]
          |
[Control Layer (PLC/SCADA)]
```

## 키워드
* 스마트 팩토리, 가동 상태, 품질 관리, 생산 이력, 실시간 통제

## 암기법
* 자공품대성 (자원관리, 공정관리, 품질관리, 데이터수집, 성과분석)
