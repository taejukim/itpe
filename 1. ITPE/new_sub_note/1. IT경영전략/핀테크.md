## 정의
IT 기술과 금융 서비스의 융합을 통한 혁신적인 금융 서비스, 핀테크
- Finance(금융)와 Technology(기술)의 합성어로, 모바일, 빅데이터, AI 기술을 금융에 적용
- 복잡한 금융 절차를 간소화하고 접근성을 높여 새로운 금융 경험과 가치 제공

## 구성요소
- 지급 결제: 모바일 간편 결제, 송금 서비스 (삼성페이, 토스 등)
- 금융 데이터 분석: 마이데이터, 신용 평가 모델 고도화, 로보어드바이저
- 자산 관리: 개인별 맞춤형 포트폴리오 제안 및 투자 자동화
- 자금 조달: P2P 대출, 크라우드 펀딩, 온라인 뱅킹
- 규제 샌드박스: 혁신 서비스의 시장 출시를 위해 일시적으로 규제를 면제해주는 제도

## 구성도 요소
* 에코시스템: 금융기관 + 테크기업 + 규제기관 + 소비자
* 관계: 기존 금융권(Traditional Finance)의 [[디지털 전환(DX)]]과 테크핀(Techfin)의 경쟁 및 협력
```
[Traditional Finance] <--- (Open API) ---> [Fintech Startup]
          |                                      |
          +---------- [마이데이터/혁신금융] ----------+
```

## 키워드
* 테크핀, 마이데이터, 오픈뱅킹, 규제 샌드박스, 블록체인 결제

## 암기법
