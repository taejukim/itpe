## 정의
다각적 관점의 균형 잡힌 전략 성과 관리 체계, BSC
- 재무적 지표 외에 고객, 내부 프로세스, 학습과 성장 관점을 통합하여 관리하는 기법
- 조직의 비전과 전략을 측정 가능한 목표(KPI)로 전환하여 전략 실행력 강화

## 구성요소
- 재무적 관점: 주주 가치 극대화, 수익성, 매출 성장 (과거 성과)
- 고객 관점: 고객 만족도, 시장 점유율, 신규 고객 확보 (외부 인식)
- 내부 프로세스 관점: 운영 효율성, 혁신 사이클, 품질 관리 (내부 활동)
- 학습과 성장 관점: 직원 역량, 정보 시스템 활용도, 조직 문화 (미래 동력)

## 구성도 요소
* 인과 관계: 학습/성장 → 내부 프로세스 → 고객 만족 → 재무 성과 (전략 맵)
* 상시 관리: 비전/전략 → 전략 목표 → KPI 설정 → Target 설정 → Initiative(실행 방안)
```
      [재무 관점]
         ^
      [고객 관점]
         ^
    [내부 프로세스]
         ^
    [학습과 성장]
```

## 키워드
* KPI, CSF, 전략 맵(Strategy Map), 균형(Balanced), 인과관계

## 암기법
* 재고내학 (재무, 고객, 내부프로세스, 학습성장)
