## 정의
공간에 녹아든 디지털 기술을 통한 비가시적 상거래 경험, 엠비언트 커머스
- IoT, AI, 센서 기술을 활용하여 사용자가 의식하지 못하는 사이에 상거래가 일어나는 환경
- 물리적 매장이나 주거 공간 자체가 쇼핑 채널이 되는 '생활 밀착형' 커머스

## 구성요소
- 환경 지능(Ambient Intelligence): 센서를 통해 사용자의 맥락과 주변 상황을 실시간 파악
- 무마찰 결제(Frictionless Payment): 결제 과정을 생략하거나 자동화 (예: 아마존 고)
- 개인화 추천: 위치 및 행동 패턴 데이터를 분석하여 실시간으로 최적의 상품 제안
- 비가시적 UI: 음성 명령, 동작 인식 등 별도의 단말 조작이 필요 없는 인터페이스
- 자동 보충(Auto-replenishment): 생필품 소진 시 자동으로 주문 및 배송 수행

## 구성도 요소
* 아키텍처: 물리적 공간(IoT 센서) → 클라우드 분석(AI) → 자동 서비스(커머스)
* 관계: O2O(Online to Offline)를 넘어선 온-오프라인의 완전한 융합(Phygital)
```
[User Context] -- (Sensor Data) --> [Contextual Intelligence]
       ^                                     |
       |                               (Auto Transaction)
       +------- [Seamless Experience] <------+
```

## 키워드
* Context-Awareness, 비가시적 경제, 무마찰 결제, 아마존 고, 환경 지능

## 암기법
