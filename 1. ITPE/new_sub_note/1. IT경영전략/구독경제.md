## 정의
일정 금액을 지불하고 주기적으로 제품이나 서비스를 이용하는 경제 모델, 구독경제
- '구매'에서 '구독'으로 소비 패러다임이 전환된 모델로, 정기적인 이용권을 기반으로 함
- 클라우드 SaaS, 콘텐츠 스트리밍(넷플릭스), 가전 렌탈, 정기 배송 등 다양한 분야로 확장

## 구성요소
- 정기 결제: 월간/연간 단위의 고정적인 비용 지불 체계
- 큐레이션(Curation): 사용자 데이터 기반의 개인 맞춤형 서비스 제공
- 락인 효과(Lock-in): 장기 이용을 통한 고객 이탈 방지 및 충성도 강화
- 무제한 접근: 정해진 기간 내에 서비스를 자유롭게 이용하는 방식
- 관계 마케팅: 단발성 판매가 아닌 고객과의 지속적인 연결 및 관리

## 구성도 요소
* 가치 사슬: 고객 데이터 수집 → 패턴 분석 → 개인화 추천 → 구독 유지 → 피드백
* 관계: 소유의 부담을 줄이고 경험의 가치를 높이는 방향으로 진화
```
[User] ---- (Monthly Fee) ----> [Service Provider]
   ^                                   |
   +--- (Continuous Value Delivery) ---+
   |                                   |
   +--- (Data-driven Curation) <-------+
```

## 키워드
* SaaS, 넷플릭스 모델, 큐레이션, 락인 효과, 지속적 가치
