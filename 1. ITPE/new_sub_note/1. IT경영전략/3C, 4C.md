## 정의
핵심 이해관계자 분석을 통한 전략 수립 기법, 3C/4C
- 오마에 겐이치가 제안한 기법으로 고객(Customer), 경쟁사(Competitor), 자사(Company)를 분석
- 거시환경인 환경(Circumstance)을 추가하여 4C로 확장, KSF(핵심성공요인) 도출에 활용

## 구성요소
- Customer(고객): 시장 규모, 성장성, 고객 니즈, 구매 행동 분석
- Competitor(경쟁사): 경쟁 현황, 경쟁사 강약점, 시장 점유율, 수익성 분석
- Company(자사): 핵심 역량, 브랜드 파워, 재무 능력, 프로세스 분석
- Circumstance(환경): PEST(정치, 경제, 사회, 기술) 등 거시적 외부 환경 분석

## 구성도 요소
* 분석 절차: 시장 정의 → 3C/4C 분석 → KSF 도출 → 전략 수립
* 관계: 고객 니즈와 경쟁사 대비 자사 역량의 교집합에서 차별화 포인트 도출
```
      [Competitor] <-----> [Company]
            \               /
             \   (Strategy) /
              \           /
               [Customer]
                   |
            [Circumstance]
```

## 키워드
* KSF(핵심성공요인), PEST, 미충족 니즈, 핵심 역량, 경쟁 우위

## 암기법
* 고경기환 (고객, 경쟁사, 기업, 환경)
