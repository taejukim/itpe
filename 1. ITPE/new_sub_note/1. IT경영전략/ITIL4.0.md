## 정의
사용자 가치 창출 중심의 IT 서비스 관리 베스트 프랙티스, ITIL 4
- IT 서비스를 공동 가치 창출(Co-creation)의 관점에서 정의한 서비스 관리 프레임워크
- 기존 프로세스 중심에서 실천 방식(Practices)과 가치 흐름(Value Streams) 중심으로 패러다임 전환

## 구성요소
- SVS(Service Value System): 조직의 모든 활동이 가치로 연결되는 전체 시스템
- SVC(Service Value Chain): 가치 창출을 위한 핵심 활동 모델 (Plan, Improve, Engage 등)
- 4 Dimensions: 조직/사람, 정보/기술, 파트너/공급자, 가치 흐름/프로세스
- 7 Guiding Principles: 가치 집중, 현 상태 유지, 점진적 반복, 협업/투명성 등
- Practices: 34가지 관리 실천 방식 (General, Service, Technical)

## 구성도 요소
* 아키텍처: 기회/수요(Opportunity/Demand) → SVS(SVC 포함) → 가치(Value)
* 핵심 루프: 서비스의 지속적 개선(Continual Improvement) 강조
```
[Demand] -> [Service Value System] -> [Value]
               |
        [Service Value Chain]
 (Plan -> Engage -> Design -> Obtain -> Deliver -> Improve)
```

## 키워드
* 가치 창출(Value), SVC, SVS, 4차 산업혁명 대응, Agile/[[DevOps]] 통합
