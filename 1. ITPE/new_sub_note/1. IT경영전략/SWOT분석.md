## 정의
내부 역량과 외부 환경을 결합한 전략 수립 분석 도구, SWOT
- 조직의 내부적 강점(S)·약점(W)과 외부적 기회(O)·위협(T) 요인을 분석하는 기법
- 분석 결과를 바탕으로 최적의 비즈니스 전략(SO, ST, WO, WT)을 도출함

## 구성요소
- Strength(강점): 조직이 보유한 우수한 역량, 자원, 브랜드 파워
- Weakness(약점): 경쟁사 대비 취약한 점, 부족한 리소스, 내부 문제
- Opportunity(기회): 성장 가능성, 정부 정책 변화, 새로운 기술 트렌드
- Threat(위협): 경쟁 심화, 경기 침체, 법적 규제 강화, 대체재 출현

## 구성도 요소
* 전략 매트릭스:
    - SO(공격적): 강점을 활용하여 기회 선점
    - ST(다양화): 강점을 활용하여 위협 극복
    - WO(방향전환): 약점을 보완하여 기회 활용
    - WT(방어적): 약점을 보완하고 위협 회피
```
          [Internal] | [Strength] | [Weakness]
[External]-----------+------------+-----------
[Opportunity]        |   SO 전략  |   WO 전략
---------------------+------------+-----------
[Threat]             |   ST 전략  |   WT 전략
```

## 키워드
* 환경 분석, 전략 도출, SO/ST/WO/WT, 내부 역량, 외부 환경

## 암기법
* 강약기위 (강점, 약점, 기회, 위협)
* 공다전방: 공격적(SO), 다양화(ST), 방향전환(WO), 방어적(WT)
