## 정의
대량의 데이터에서 숨겨진 패턴과 가치를 추출하는 기법, 데이터 마이닝
- 방대한 데이터베이스 내에 존재하는 유의미한 상관관계나 패턴을 발견하여 의사결정에 활용
- 통계학, AI, 머신러닝 기법을 융합하여 미래를 예측하고 비즈니스 통찰력 제공

## 구성요소
- 분류(Classification): 의사결정 나무, 베이즈 분류 등을 통한 데이터 범주화
- 군집(Clustering): 유사한 특성을 가진 데이터들을 그룹으로 묶음 (K-Means 등)
- 연관(Association): 장바구니 분석과 같이 항목 간의 관련성 발견 (Apriori 등)
- 연속(Sequence): 시간적 선후 관계에 따른 패턴 발견
- 회귀(Regression): 변수 간의 관계를 수식으로 모델링하여 연속값 예측

## 구성도 요소
* 수행 단계(CR[[ISP]]-DM): 비즈니스 이해 → 데이터 이해 → 데이터 준비 → 모델링 → 평가 → 전개
* 관계: 데이터 웨어하우스(DW) 또는 데이터 레이크에서 정제된 데이터를 대상으로 분석 수행
```
[Raw Data] -> [DW] -> [Pre-processing] -> [Mining] -> [Pattern/Knowledge]
                                            ^              |
                                            +--- [Feedback]-+
```
