## 정의
IT 투자의 경제적 가치를 분석하고 평가하는 정량적 분석 기법, 경제성 평가
- 정보화 사업의 비용 대비 편익을 산출하여 사업 추진 여부를 결정하는 의사결정 도구
- 화폐의 시간적 가치를 고려하여 객관적인 투자 타당성 근거를 제공함

## 구성요소
- ROI(Return on Investment): (순이익 / 투자 비용) x 100, 가장 보편적인 지표
- NPV(Net Present Value): 미래 수익의 현재 가치 합계에서 투자 비용을 뺀 값 (0보다 커야 함)
- IRR(Internal Rate of Return): NPV를 0으로 만드는 할인율 (기준 수익률보다 커야 함)
- PBP(Payback Period): 투자 원금을 회수하는 데 걸리는 기간
- B/C Ratio(Benefit-Cost Ratio): 편익의 현재 가치를 비용의 현재 가치로 나눈 비율 (1보다 커야 함)

## 구성도 요소
* 평가 프로세스: 비용/편익 항목 식별 → 미래 가치 추정 → 현재 가치 환산 → 지표 산출
* 관계: TCO(비용) 산정과 현금 흐름(Cash Flow) 분석이 선행되어야 함
```
[비용 추정(TCO)] <-> [편익 추정(Benefit)]
          |                 |
    [화폐의 시간적 가치 할인]
          |
    [경제성 지표 산출 (NPV, IRR 등)]
```

## 키워드
