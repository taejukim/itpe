## 정의
IT 투자의 가치를 정량적·정성적으로 측정하는 성과 평가 모델, IT-ROI
- 정보화 투입 비용 대비 창출된 경제적·전략적 효과를 산출하여 투자의 타당성을 검증하는 체계
- 기획, 구축, 운영의 전 생애주기에 걸쳐 성과를 관리하여 IT 자산의 가치를 극대화함

## 구성요소
- 정량적 지표: NPV(순현재가치), IRR(내부수익률), PBP(회수기간), ROI(투자수익률)
- 정성적 지표: 전략적 정렬도, 고객 만족도, 브랜드 가치, 직원 역량 향상
- TCO(Total Cost of Ownership): 도입부터 폐기까지의 총 소요 비용 분석
- TVO(Total Value of Opportunity): 무형적 가치를 포함한 총 기회 가치
- 측정 단계: 사전 평가(타당성), 중간 평가(진척도), 사후 평가(목표 달성도)

## 구성도 요소
* 측정 기법: 정량적 평가(경제성 지표) + 정성적 평가([[BSC]], 다기준 분석)
* 관계: TCO(비용)와 TVO(편익)를 비교하여 최종 ROI 산출
```
[Investment Planning] -> [Execution] -> [Operation]
         |                  |              |
    (Ex-ante)           (Mid-term)     (Ex-post)
         \                  |             /
          +---- [IT-ROI 성과 분석 체계] ----+
```

## 키워드
* TCO, NPV, IRR, [[BSC]] 연계, 무형 가치 산정

## 암기법
* 사중후 (사전, 중간, 사후 평가)
