#정보관리기술사
## 정의 
SCM (Supply Chain Management)
Value Chain의 통합관리, SCM 개요
- 공급자에서 소비자까지 제품 공급 사슬의 전 과전을 효율적으로 통합관리하는 정보화 경영혁신방법

## 목적
- value chain 통합
- 비용 절감
- 수익 증대
- 효율성 증대

## 구성도
- 공급자 → 구매 시스템 → 생산 시스템/재고 시스템 → 판매 유통 시스템 → 소비자
- 공급자 ~ 생산시스템/재고시스템  : SCP 자원운영(계획, 통제)
- 생산시스템/재고시스템 ~ 소비자 : SCE, 물류, 수송
- 전체 단계 : 공급망 프로세스 관리 (SCPM)

## 구성요소
- SCP(Planning) : 제조, 수요, 유통, 운송
- SCE(Execution) : 주문 관리, 생산 관리, 유통 관리, 역물류 관리
- SCI(Integration) : 모니터링, 성과 측정, 프로세스 개선
- SCPM(P)
