## 프로토타이핑 모델

#flashcard
프로토타이핑 모델이란?
프로토타이핑 모델
고객 요구사항이 불명확할 때 초기 모형을 제작하여 고객과 의사소통을 하여 조정하는 개발 모델, 프로토타이핑 모델 빠르게 프로토타입을 만들어 고객 피드백을 받아 요구사항을 명확히 하는 반복적 접근법 요구사항이 불명확하거나 시스템 이해도가 낮은 경우에 효과적인 개발 방법론

#flashcard
프로토타이핑 모델의 핵심 키워드는?
프로토타입, 요구사항 불명확, 의사소통, 피드백, 반복적 개발, 폭포수 모델, Spiral 모델


#flashcard
프로토타이핑 모델의 암기법은?
프로토타이핑 모델
프로요설개고피: 프로토타입 정의, 요구사항 수집, 설계, 개발, 고객 평가, 피드백
