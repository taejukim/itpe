## 연동계획(Rolling Wave Planning)

#flashcard
연동계획(Rolling Wave Planning)이란?
연동계획(Rolling Wave Planning)
가까운 미래는 상세하게, 먼 미래는 개괄적으로 계획하는 점진적 계획 기법, Rolling Wave Planning 프로젝트 초기에는 불확실성이 높아 먼 미래의 계획을 상세히 수립하기 어려울 때 사용 시간이 지나면서 정보가 명확해질수록 계획을 점진적으로 상세화하는 반복적 계획 방법

#flashcard
연동계획(Rolling Wave Planning)의 핵심 키워드는?
점진적 계획, 근접 계획, 원거리 계획, 계획 갱신, 반복적 개선


#flashcard
연동계획(Rolling Wave Planning)의 암기법은?
연동계획(Rolling Wave Planning)
근원갱반: 근접 계획, 원거리 계획, 계획 갱신, 반복적 개선
