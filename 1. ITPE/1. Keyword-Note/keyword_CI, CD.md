## CI, CD

#flashcard
CI, CD이란?
CI, CD
지속적 통합과 지속적 배포를 통한 소프트웨어 개발 자동화, CI/CD 코드 변경을 자동으로 통합, 테스트, 배포하여 개발 효율성을 향상시키는 방법론 수동 작업을 최소화하고 배포 주기를 단축하여 고품질 소프트웨어를 빠르게 전달 DevOps의 핵심 구성요소로, Agile 프로세스 XP의 지속적 통합 개념을 확장

#flashcard
CI, CD의 핵심 키워드는?
CI, CD, 자동화 파이프라인, 버전 관리, 자동 빌드, 자동 테스트, 자동 배포


#flashcard
CI, CD의 암기법은?
CI, CD
통배자버자: 통합(CI), 배포(CD), 자동화 파이프라인, 버전 관리, 자동 테스트
