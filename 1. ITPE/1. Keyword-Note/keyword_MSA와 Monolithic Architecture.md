## MSA와 Monolithic Architecture

#flashcard
MSA와 Monolithic Architecture이란?
MSA와 Monolithic Architecture
마이크로서비스 아키텍처와 모놀리식 아키텍처의 비교, MSA vs Monolithic Monolithic: 모든 기능을 하나의 애플리케이션으로 구성하는 전통적 아키텍처 MSA: 애플리케이션을 독립적인 작은 서비스로 분해하는 현대적 아키텍처

모놀리식, 마이크로서비스, 독립적 배포, 약한 결합, 확장성, API Gateway, MSA 서비스 서비스 메쉬(Service Mesh)
