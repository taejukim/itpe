## 정보은닉(Information Hiding)

#flashcard
정보은닉(Information Hiding)이란?
정보은닉(Information Hiding)
객체의 내부 구현 세부사항을 숨기고 외부에는 인터페이스만 노출하는 객체지향 원칙, 정보은닉 객체의 내부 데이터 구조와 구현 방법을 외부에 숨겨 결합도를 낮추는 기법 인터페이스를 통해서만 객체와 상호작용하도록 하여 유지보수성과 재사용성을 향상

#flashcard
정보은닉(Information Hiding)의 핵심 키워드는?
구현 은닉, 인터페이스 노출, 접근 제어, 캡슐화, 결합도 감소

정보은닉(Information Hiding)
