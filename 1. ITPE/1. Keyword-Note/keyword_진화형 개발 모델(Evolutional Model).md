## 진화형 개발 모델(Evolutional Model)

#flashcard
진화형 개발 모델(Evolutional Model)이란?
진화형 개발 모델(Evolutional Model)
시스템의 프로토타입을 개발하면서 지속적으로 발전시켜 나가는 방법, 진화형 개발 모델 초기 프로토타입에서 시작하여 사용자 피드백을 받아 지속적으로 기능을 추가하고 개선하는 방법론 진화에 대한 전체 개요(Outline)가 필요하며, 핵심 기능 개발 후 구성요소를 개선

#flashcard
진화형 개발 모델(Evolutional Model)의 핵심 키워드는?
프로토타입, 지속적 발전, 진화, 피드백, 점진적 개선


#flashcard
진화형 개발 모델(Evolutional Model)의 암기법은?
진화형 개발 모델(Evolutional Model)
전프사기구지: 전체 개요, 프로토타입 개발, 사용자 피드백, 기능 추가, 구성요소 개선, 지속적 진화
