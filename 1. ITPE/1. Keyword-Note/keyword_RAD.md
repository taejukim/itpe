## RAD

#flashcard
RAD이란?
RAD
강력한 소프트웨어 개발 도구를 이용하여 매우 짧은 주기로 개발을 진행하는 순차적 소프트웨어 개발 프로세스, Rapid Application Development CASE 도구를 활용하여 빠른 개발 기간(60-90일) 내에 시스템을 개발하는 방법론 사용자의 적극적인 참여를 통해 요구사항을 빠르게 반영하고 프로토타입을 통해 검증

#flashcard
RAD의 핵심 키워드는?
JRP, JAD, CASE 도구, 프로토타입, 사용자 참여, 빠른 개발
