## 상속성(Inheritance)

#flashcard
상속성(Inheritance)이란?
상속성(Inheritance)
기존 클래스의 속성과 메서드를 상속받아 새로운 클래스를 만드는 객체지향 원칙, 상속성 상위 클래스(부모 클래스)의 특성을 하위 클래스(자식 클래스)가 물려받아 재사용하는 기법 코드 중복을 제거하고 계층 구조를 통해 클래스 간 관계를 명확히 표현

#flashcard
상속성(Inheritance)의 핵심 키워드는?
상위 클래스, 하위 클래스, 메서드 상속, 오버라이딩, 다중 상속, 계층 구조, 객체지향 프로그래밍, 다형성(Polymorphism)

상속성(Inheritance)의 암기법은?
상속성(Inheritance)
상하메오다계: 상위 클래스, 하위 클래스, 메서드 상속, 오버라이딩, 다중 상속, 계층 구조
