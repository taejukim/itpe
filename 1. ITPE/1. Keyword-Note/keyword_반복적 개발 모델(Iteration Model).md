## 반복적 개발 모델(Iteration Model)

#flashcard
반복적 개발 모델(Iteration Model)이란?
반복적 개발 모델(Iteration Model)
사용자의 요구사항이나 개발 범위를 분해하여 점증적으로 반복 개발하여 최종 시스템으로 완성하는 개발 모델, 반복적 개발 모델 전체 시스템을 작은 단위로 나누어 반복적으로 개발하여 점진적으로 완성하는 방법론 각 반복마다 완전한 기능을 구현하여 점진적으로 시스템을 확장

#flashcard
반복적 개발 모델(Iteration Model)의 핵심 키워드는?
반복적, 점증적, 분해, 통합, 점진적 완성, Spiral 모델, Agile 프로세스 Scrum


#flashcard
반복적 개발 모델(Iteration Model)의 암기법은?
반복적 개발 모델(Iteration Model)
분반설테통검: 분해, 반복 계획, 설계/개발, 테스트, 통합, 검증
