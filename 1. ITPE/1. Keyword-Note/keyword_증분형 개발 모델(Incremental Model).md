## 증분형 개발 모델(Incremental Model)

#flashcard
증분형 개발 모델(Incremental Model)이란?
증분형 개발 모델(Incremental Model)
요구사항의 범위 일부분을 반복적으로 개발하며, Speed를 위해 병렬 개발하면서 대상 범위를 개발하여 최종 제품을 완성하는 방법, 증분형 개발 모델 폭포수 모델의 변형으로 하나의 시스템을 여러 서브시스템으로 분리하여 점차 증가시키는 방법론 각 증분(Increment)은 독립적으로 개발되고 통합되어 최종 시스템을 완성

#flashcard
증분형 개발 모델(Incremental Model)의 핵심 키워드는?
증분, 병렬 개발, 점진적 증가, 독립적 개발, 통합


#flashcard
증분형 개발 모델(Incremental Model)의 암기법은?
증분형 개발 모델(Incremental Model)
전증개통검: 전체 설계, 증분 정의, 증분 개발, 통합, 검증
